## 本海域の特徴

-   艦これ第二期に伴って新規実装された海域です。出撃画面では北方海域と西方海域の間に表示される、新たな海域群となります。
-   実装当初、当ウィキでは南西海域を4番目の海域として扱っていましたが、8月18日19時40分頃、公式から「南西海域：第1海域を#7-1と呼称する旨」が告知されました。  
    それに伴い、当ウィキでも南西海域を7番目の海域として扱っています。報告の際はご注意ください。
-   2018年8月17日(第二期開始日)に7-1、2018年11月16日に7-2、2020年9月17日に7-3、2021年12月28日に7-4が開放されました。  
    2023年1月20日には、6-5以来約6年3か月ぶりとなるEO・7-5が実装されました。
-   2021年12月28日のアップデートで、同日追加の7-4に基地航空隊を出すことが可能になりました。詳細は個別ページ参照。
-   中部海域に続く7番目の海域ですが、難易度は海域によって大きく異なります。
    -   7-1は5隻の水雷戦隊で攻略する対潜マップです。  
        序盤から攻略必須ということもあってか難易度は低めです。
    -   7-2は近年のイベント海域で見られる**複数本ゲージのチュートリアル**的なマップです。  
        ゲージ1本目は対潜戦、ゲージ2本目は水上艦隊戦がメインです。
    -   7-3は道中・ボス共に非常に弱いものの、持ち込める戦力が極めて限られている3-2のようなマップです。  
        **史実艦によるルート固定**が攻略において非常に重要です。
    -   7-4は道中・ボスに**強力な潜水艦が多数出現**する、対潜マップの集大成のような高難易度マップです。  
        ボス本体が非常に硬く、イベント海域同様戦力の配分が重要になります。
    -   7-5は**通常海域初の3本ゲージ、さらに最終ゲージの出現にギミックマスの攻略が必要となる、近年のイベント海域のチュートリアル**的なマップです。  
        道中は比較的弱く、各ゲージも２～３回でクリア可能と難易度は高くありませんが、基地航空隊や支援艦隊は使用できないため、攻略には艦隊の地力が求められます。
-   7-2～7-5はEO同様、**月始めにゲージが復活する**という特徴があります。
    -   なお、1度でもクリアしていれば次月以降も次の海域には出撃可能です(1-6のように月ごとに1-5をクリアし直す必要はありません)。

## 7-1. ブルネイ泊地沖

### 1. 海域概要・特徴
海域全体が対潜戦メインとなっており、燃料および戦果稼ぎ、ならびに軽巡・駆逐・海防艦のレベリングで極めて人気のある定番の海域です。
*   **作戦名**: ブルネイ泊地沖哨戒
*   **難度**: ☆☆☆☆☆
*   **クリア条件**: ボス（Kマス）旗艦の3回撃破（戦力ゲージクリア後は復活なし）
*   **特徴**:
    *   道中およびボスのすべての戦闘が対潜戦です。こちらの被ダメージをほぼゼロにするために、**対潜先制爆雷攻撃**が可能な艦を複数編成することが必須です。
    *   道中のEマスで燃料が獲得できるため、燃費に優れた艦隊で周回することで、資材を増やしながら戦果と経験値を稼ぐことができます。

---

### 2. ルート分岐法則
艦隊内の隻数、艦種構成、および特定の艦（潜水艦・戦艦・空母など）の有無によってルートが決定されます。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **スタート** | ・潜水艦を含む場合:<br>　・(戦艦級+空母系)1隻以上、または5隻以上の編成 → **BまたはDのランダム**<br>　・4隻以下の編成 → **ランダム**<br>・(戦艦級+空母系)1隻以上、または6隻編成 → **B**（通常）<br>・5隻編成、または補給艦を含む → **D**（対潜）<br>・4隻以下の編成 → **F**（対潜・最短） |
| **B** | ・(戦艦級+正規空母)1隻以上、軽空母2隻以上、または重巡級3隻以上 → **A**（うずしお）<br>・(駆逐+海防)2隻以上 → **C**（通常）<br>・それ以外は **A/Cのランダム** |
| **D** | ・[軽巡1 ＋ 駆逐4] または [駆逐1以上 ＋ 海防3以上] または [補給1以上 ＋ 海防3以上] → **E**（燃料）<br>・(駆逐+海防)5隻、または[(駆逐+海防)4隻 ＋ (練巡or補給)1] → **E**（燃料）<br>・[(駆逐+海防)4隻 ＋ 水母1] → **CまたはEのランダム**<br>・それ以外は **C**（通常） |
| **H** | ・[軽巡1以上 ＋ 駆逐4以上] または [駆逐1以上 ＋ 海防3以上] → **K（ボス）**<br>・補給艦1隻以上 → **IまたはKのランダム**<br>・(戦艦級+空母系)2隻以上 → **J**（通常）<br>・(戦艦級+空母系)1隻 → **ランダム**<br>・それ以外は **I/J/Kのランダム** |

*   **推奨周回ルート【EDHK】**: D（対潜） → E（燃料） → H（対潜） → K（ボス）
*   **軽量レベリングルート【FGK】**: F（対潜） → G（通常） → K（ボス）（※4隻編成以下で進行可能）

---

### 3. おすすめ編成例
#### ◆ ブルネイ周回・戦果稼ぎ編成（推奨・定番）
*   **編成**: 軽巡1 ＋ 駆逐4
*   **ルート**: D → E（燃料） → H → K（ボス）（道中2戦・燃料獲得）
*   **陣形**: D:単横陣 / H:単横陣 / ボス（K）:単横陣
*   **装備**:
    *   全員に**対潜先制爆雷攻撃**（対潜値100以上＋ソナー＋爆雷投射機＋爆雷等のシナジー）を装備。
    *   軽巡（夕張改二特、矢矧改二乙など）には甲標的を装備させて道中の撃ち漏らし対策（CやGに逸れた場合のケア）や、ソナーを満載して対潜火力を最大化します。
*   **特徴**: 艦これ随一の戦果周回・資材稼ぎ編成です。道中の被弾率が非常に低く、バケツ（高速修復材）の消費を極めて低く抑えながら高速で周回・レベリングが可能です。

#### ◆ 海防艦レベリング編成（超低燃費）
*   **編成**: 駆逐1 ＋ 海防艦4（または 軽巡1 ＋ 海防艦3）
*   **ルート**: D → E（燃料） → H → K（ボス）（道中2戦）
*   **特徴**: 消費資材が極めて少ない海防艦をメインとしたレベリング編成。対潜先制爆雷攻撃が発動するLvまで上がった海防艦を複数並べることで、ノーダメージでボスS勝利を量産できます。

### 6. 特記事項・ギミック
*   **対潜先制爆雷攻撃の条件**:
    *   通常は「ソナーを装備し、かつ対潜値が100以上の艦」が先制対潜攻撃を行います（五十鈴改二、龍田改二、英国駆逐艦、無条件先制の海防艦などは対潜値100未満でも可能）。
    *   「ソナー ＋ 爆雷投射機 ＋ 爆雷」などの対潜シナジー装備をセットにすることで、敵潜水艦を一撃で撃沈しやすくなります。
*   **資源ボーナス**: Eマスの燃料獲得量は、ドラム缶（1個につき+2）や大発動艇（1個につき+3）などの輸送系装備を搭載することで獲得量をアップさせることができます。

## 7-2. タウイタウイ泊地沖

### 1. 海域概要・特徴
通常海域で初めて戦力ゲージを2本備える海域です。毎月1日0時にゲージが両方復活し、マップ進行ルートも初期化されます（勲章などのクリア報酬はありません）。
*   **作戦名**: セレベス海戦闘哨戒
*   **難度**: ☆☆☆☆☆☆☆☆
*   **クリア条件**:
    *   **第一ゲージ（Gマス・ボス1）**: 旗艦の3回撃破で破壊。破壊後、マップ右半分と第二ボスマスが出現します。
    *   **第二ゲージ（Mマス・ボス2）**: 旗艦の4回撃破で海域クリア。
*   **特徴**:
    *   第一ゲージは対潜戦メイン、第二ゲージは敵空母機動部隊（ヲ級改、ヌ級改など）との通常戦・防空戦が中心になります。
    *   第二ゲージクリア後は、ボスマス（Mマス）の敵編成が弱パターン（制空値が大幅低下）で固定されます。

---

### 2. ルート分岐法則
隻数、特定の艦種（戦艦、空母）、速力、および索敵スコア（分岐点係数4）によって決定されます。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **スタート** | ・(駆逐+海防)1隻以下、または潜水艦を含む → **A**（うずしお）<br>・【6隻編成】正規空母2隻以上、または(戦艦+正規空母)4隻以上、または軽巡級3隻以上 → **A**（うずしお） / それ以外は **B**（通常）<br>・【5隻編成】正規空母3隻以上 → **A**（うずしお） / (戦艦+正規空母)1隻以上、または軽巡級2隻以上、または海防2隻以下 → **B**（通常） / それ以外は **C**（対潜）<br>・【4隻以下】(戦艦+正規空母)1隻以上、または(駆逐+海防)2隻以下 → **B**（通常） / それ以外は **C**（対潜） |
| **C** | ・補給艦含む、または潜水艦含む → **D**（通常・気のせい）<br>・【6隻】(戦艦+正規空母)1隻以上 → **D**（通常） / (駆逐+海防)4隻以上 → **E**（対潜） / それ以外は **D**（通常）<br>・【5隻】(戦艦+正規空母)2隻以上 → **D**（通常） / (駆逐+海防)4隻以上、または海防3隻以上 → **E**（対潜） / それ以外は **D**（通常）<br>・【4隻以下】(戦艦+正規空母)2隻以上 → **D**（通常） / (駆逐+海防)3隻以上、または海防2隻以上 → **E**（対潜） / それ以外は **D**（通常） |

*   **第一ゲージ最短ルート【CEG】**: 5隻編成以下（(駆逐+海防)3〜4隻 ＋ 自由枠1〜2隻）

### 3. おすすめ編成例
#### ◆ 第一ゲージ：対潜最短編成（推奨・あ号消化/レベリング）
*   **編成**: 軽空母1（護衛空母） ＋ 海防艦3 ＋ 軽巡（または駆逐/雷巡）1
*   **ルート**: C（対潜） → E（対潜） → G（第一ボス）（道中2戦）
*   **陣形**: C:単横陣 / E:単横陣 / ボス（G）:複縦陣または単横陣（※ボスに水上艦が混じるため、先制対潜が十分なら複縦陣推奨）
*   **特徴**: 護衛空母の開幕爆撃と海防艦の先制対潜で、安全かつ極めて低い消費資材でボスを撃破・周回できます。第一ボス（Gマス）はウィークリー任務の「あ号作戦」におけるボスマス判定になります。

#### ◆ 第二ゲージ：高速統一最短編成（推奨・定番）
*   **編成**: 戦艦1 ＋ 正規空母1 ＋ 軽空母1 ＋ 航巡（または軽巡）1 ＋ 駆逐2
*   **ルート**: B → C（対潜） → D → I → M（第二ボス）（道中3戦・高速統一）
*   **陣形**: B:単縦陣 / C:単横陣 / I:単縦陣 / ボス（M）:単縦陣
*   **制空値**: 360〜390程度（ボスのヲ級改編成に対して航空優勢を確保）
*   **特徴**: バランス良く道中・ボスの火力を維持した基本攻略編成です。戦艦を組み込むことで砲撃戦が2巡化し、撃ち漏らしによる事故を大幅に減らせます。

### 2. ルート分岐法則
艦隊構成（戦艦、空母の隻数）、速力、および索敵スコア（分岐点係数4）によって決定されます。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **スタート** | ・高速+以上統一、または(正規空母1以下かつ軽空2以下かつ戦艦2以下かつ重巡2以下かつ軽巡1以上かつ(駆逐+海防)3以上) → **D**（通常）<br>・それ以外は **C**（通常）へ逸れる |
| **D** | ・最速統一、または高速+統一 → **F**（能動分岐）<br>・正規空母2隻以上、空母系3隻以上、または(戦艦級+正規空母+重巡級)3隻以上 → **E**（PT小鬼）<br>・戦艦級2隻の場合かつ[(高速統一かつ(駆逐+海防)2以上)または(駆逐+海防)3以上] → **F**（能動分岐）<br>・(軽巡+駆逐+海防)2隻以上 → **F**（能動分岐）<br>・それ以外は **E**（PT小鬼） |
| **I** | ・【索敵スコアチェック（分岐点係数4）】<br>　・索敵スコア **59以上** で **M**（通常）進行、52以下でL（通常） |
| **P（第三出現後）** | ・索敵スコア **63以上** でかつ「正規空母0、(戦艦+軽空母)1以下、(戦艦+重巡級)2以下、軽巡1以上」 → **T（第三ボス）**<br>・最速統一 → **T（第三ボス）**<br>・それ以外は **R**（通常・逸れ） |

*   **最短ルート【A-B-D-F-J-O-Q(またはT)】**: A（空襲） → B（対潜） → D（水上） → F（能動） → J（水上） → O（能動） → 各ボス
