## 本海域の特徴

-   2013/09/18に5-1,5-2、10/04に5-3、2014/02/26に5-4、2014/03/14に5-5(EO)、2026/05/29に5-6(EO)が実装されました。
-   ルート固定や潜水艦、ボス戦力ゲージが登場する西方海域をクリアした提督向けということなのか、**ここから難易度が急上昇します。**
-   総じて、海域攻略には大型・小型問わず様々な艦種で、全体の練度の底上げ、上位装備の用意などが必要となってきます。

-   5-x海域には、**支援艦隊**を出すことが可能です。  
    また、**二期からは夜戦マスにも支援が到着するようになりました**。
    -   この海域を対象とする各種出撃任務や、支援の働き次第で道中の安定度がガラリと変わる5-3、道中からボスまで強烈な編成が並ぶ5-5に挑戦する際は特に、**支援の有無で****難易度が大幅に変化します**。  
        道中の危険度が上がっている二期では、必要と感じたら出し惜しみせずに利用しましょう。
    -   前衛支援(道中支援)は「No18航空機輸送作戦」クリアで出現します\*1
    -   決戦支援(ボス支援)は「No26敵母港空襲作戦」クリアで出現します\*2
        -   ただし、これらの遠征をクリアするためには膨大な時間が必要となります。  
            「No25通商破壊作戦遠征」は40時間、「No26敵母港空襲作戦」に至っては80時間かかります。  
            編成ミスや補給忘れ、旗艦レベル不足等がないように十分確認して出しましょう。

-   **海域全体の注意点**
    -   5-2以降すべてのマップに戦力ゲージが存在し、複数回の海域勝利が求められます。仕様は4-4と同じくボス旗艦撃沈でのみ減少するタイプです。
    -   敵の制空値が大幅に強化されます。  
        第一期では、一部EO海域、及び最深部であった6-ｘ海域でしか出現しなかった、空母ヲ級flagshipや、ヲ級flagshipの更なる上位種である空母ヲ級改flagshipが登場します。
        -   白い新型艦載機~いわゆるたこ焼き~は、制空値だけでなく開幕攻撃もかなり強力。  
            制空権を取っていても、撃墜判定次第では小型～中型艦がいきなり中～大破するということも起こりえます。
        -   ヲ級改flagshipは、制空値だけでなく耐久性が大幅に強化され、装甲と耐久は戦艦ル級改flagshipをも上回ります。  
            戦艦の弾着カットインや空母のFBAカットインが決まってさえ、撃沈どころか中破にすらできないことも珍しくありません。  
            また、スロットあたりの搭載機数がヲ級flagshipの32機から36機へと増加したのみならず、スロット数そのものも3から4へと強化されています。  
            開幕航空戦での撃墜による攻撃機枯渇は、まず望めないと思った方が良いでしょう。
        -   対空関連のシステムはかなり複雑ですが、航空戦の項目に今一度よく目を通し、制空判定や対空関連をチェックしておきたいところです。
    -   そして、**軽巡ツ級**、**重巡ネ級**、及びその**elite**がEOを除く通常海域ではここから登場します。  
        ツ級は防空能力が非常に高く、艦載機熟練度の減少、残機0まで打ち落とされての熟練度リセットが起こりやすいので要注意。  
        また回避や雷装も高いため、砲撃戦で撃ち漏らして雷撃戦で大破をもらうということもおこり得ます。
        -   なお、この撃墜は対空砲火によるものなので、対空砲火の影響を受けない戦闘機・偵察機は撃墜が起こりません。  
            ツ級が配備されたマップでは、一桁スロットには攻撃機や爆撃機を置かない方が無難。
    -   この防空能力に秀でた艦が、道中の主要なマスに数多く鎮座します。  
        各マップとも、一期に比べ撃墜被害が大きくなりがちな点に注意。

-   以上の点から、これまでのマップに比べて空母の働きにかなり制約がかかります。  
    制空争いのために艦戦に多くのスロットを割かなければならなくなり、攻撃機はツ級の迎撃により開幕攻撃が減殺されがちです。  
    このため、砲撃戦移行までに多くの敵艦が残り易く、その後の被害拡大に繋がり易いです。
    -   水上戦闘機を搭載可能な航巡等と併用する事で、空母の制空権確保への負担を幾分か減らす事が出来ます。
    -   また、貴重なボーキサイトの消耗も全体的に多くなり、長期的な資源回復に手間取るケースも出てくるでしょう。

-   EOである5-5・5-6には、凶悪な戦闘能力を誇る**戦艦レ級**が待ち構えています。  
    攻略には万全を期し、徹底して戦力を整えましょう。
    -   次の海域である中部海域への出撃に、5-5・5-6のクリアは必要ありません。
    -   5-6は**通常海域初の輸送ゲージ**を含む**3本ゲージ**、**制空を非常に取り辛いボス**など、運営の告知通り**「ミニイベント海域」**というべき内容になっています。  
        似たような立ち位置の7-5と比べ、特に3ゲージ目のラストは随伴も強力です。上記の決戦支援の活用も検討しましょう。

## 5-1. 南方海域前面

### 1. 海域概要・特徴
南方海域の最初の海域。クリア条件はボス（Jマス）B勝利以上です。
*   **作戦名**: 南方海域進出作戦
*   **難度**: ☆☆☆☆☆☆☆☆
*   **クリア条件**: ボスB勝利以上
*   **主な入手アイテム**: 鋼材（Cマス）、弾薬（Hマス）
*   **特徴**:
    *   正規空母を入れると遠回り（E経由）になるか、ボス到達が困難になります。そのため、基本的には空母を制限した「軽量最短ルート（BCFJ）」か、水戦や航戦を活用した「重量ルート（BEGJ）」が攻略のメインとなります。
    *   難易度の高いマンスリー任務やクォータリー任務が多く指定されており、攻略の機会が多い海域です。

---

### 2. ルート分岐法則
艦隊内の「(戦艦級+空母系)」の数や、駆逐・軽巡・重巡の編成によってルートが分岐します。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **スタート** | ・(戦艦級+空母系)5隻以上 → **A**（うずしお）<br>・(戦艦級+空母系)2隻以下 かつ 駆逐2隻以上 → **B**（最短ルート）<br>・重巡級4隻以上 かつ 軽巡1隻で **B**<br>・重巡級2隻以上 かつ 軽巡1隻（過不足なく）で **B**<br>・戦艦級3隻 かつ 軽巡1隻 かつ 駆逐2隻で B寄り(75%)のランダム<br>・それ以外はランダム |
| **B** | ・正規空母を含む、または軽空母2隻以上 → **E**（対潜）<br>・戦艦級2隻以下 → **C**（鋼材）<br>・軽巡1隻（過不足なく） → **E**<br>・駆逐2隻以上 → **C**<br>・それ以外はランダム |
| **C** | **F** 確定 |
| **F** | ・(軽巡+駆逐)0隻、または(戦艦級+軽空)4隻以上 → **H**（弾薬）<br>・最速統一 → **J（ボス）**<br>・軽巡1隻以上 かつ 駆逐2隻以上 → **J**（最短固定）<br>・駆逐4隻以上 → **J**<br>・その他 → **G**（タ級×2） |
| **G** | ・(戦艦級+空母系)5隻以上 → **I**（戦闘回避）<br>・高速+統一 → **J（ボス）**<br>・駆逐4隻以上、または重巡級2以上かつ駆逐2以上、または軽巡1以上かつ駆逐2以上 → **J**<br>・それ以外はI逸れ（IからJへの進行不可） |

### 3. おすすめ編成例
#### ◆ 最短ルート軽量編成（初攻略・周回向け）
*   **編成**: 戦艦1 ＋ 軽空母1 ＋ 重巡（または航巡/雷巡）1 ＋ 軽巡1 ＋ 駆逐2
*   **ルート**: B → C（鋼材） → F → J（ボス）（道中2戦・最短固定）
*   **制空値**: 140以上（ボスのヲ級1〜2隻に対して制空拮抗〜優勢を狙う）
*   **特徴**: 道中2戦の最短ルート。正規空母はルート逸れのため使用不可。軽空母に艦戦を多めに積み、航巡や航戦（戦艦枠）に水戦を積んで制空値を補うことで安定した攻略が可能です。

#### ◆ マンスリー「水上打撃部隊」南方へ！編成
*   **編成**: 戦艦（扶桑型/伊勢型/長門型/大和型）3 ＋ 軽巡1 ＋ 自由2（重巡/航巡など）
*   **ルート**: B → E（対潜） → G（タ級×2） → J（ボス）（道中3戦、G→Jで約2割逸れあり）
*   **特徴**: 金剛型などの「高速戦艦」や海外戦艦は指定外となる点に注意。自由枠に駆逐2を入れればボス確定進行ですが道中の事故が増えます。航巡や伊勢型改二（艦戦装備可能）を投入し、水戦や艦戦で制空権を確保するのが有効です。

#### ◆ クォータリー「三川艦隊」編成
*   **編成**: (古鷹/加古/青葉/衣笠/鳥海/天龍/夕張)より4 ＋ 自由2（駆逐2推奨）
*   **ルート**: B → C → F → J（道中2戦・最短固定）
*   **特徴**: 航空戦力が一切使えず制空権喪失となるため、前衛・決戦支援を全力で出して道中を突破し、夜戦でのS勝利を狙います。

## 5-2. 珊瑚諸島沖

### 1. 海域概要・特徴
初の本格的な空母機動部隊同士の戦闘が繰り広げられるマップです。クリア条件はボス（Oマス）旗艦の4回撃沈です。
*   **作戦名**: 珊瑚諸島沖海戦
*   **難度**: ☆☆☆☆☆☆☆☆☆
*   **クリア条件**: ボス旗艦4回撃沈（戦力ゲージクリア後は復活なし）
*   **主な入手アイテム**: 弾薬（Gマス）、ボーキサイト（Jマス）
*   **特徴**:
    *   海域全体で強力な敵空母が出現するため、高い制空値の確保と防空対策（対空カットインなど）が重要になります。
    *   「五航戦（翔鶴・瑞鶴）の両方を編成」かつ「高速統一」することで、道中の敵が比較的弱いDマスを経由しつつ、戦艦を2隻持ち込める最短史実ルート（BCDFO）を通ることができます。

---

### 2. ルート分岐法則
艦隊の重量（戦艦・空母の数）や、特定の史実艦（翔鶴・瑞鶴、夕張、祥鳳など）の有無、速力によってルートが決定されます。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **スタート** | ・潜水艦を含む、または(戦艦級+空母系)5隻以上、または正規空母3隻以上 → A（うずしお）寄りのランダム<br>・それ以外は **B**（戦闘回避） |
| **C（空襲戦）** | ・**翔鶴と瑞鶴の両方を含む** かつ 【高速統一】かつ 駆逐2隻以上 → **D**（通常/対潜）<br>・空母系2隻 かつ 重巡級2隻 かつ 駆逐2隻 → **D**<br>・【低速】を含む → **E**（通常戦・タ級）<br>・戦艦級または正規空母を含む → **E**<br>・軽空母2隻 かつ 駆逐2隻以上 → **D**<br>・軽空母1隻 かつ 重巡級1隻以上 かつ 駆逐2隻以上 → **D**<br>・それ以外は **E** |
| **D（通常/対潜）** | ・史実艦（祥鳳・夕張等）の特定条件を満たす → **G**（資材ルート）<br>・それ以外は **F**（通常戦） |
| **E（通常・強）** | **F** 確定 |
| **F（通常）** | ・【索敵スコアチェック（分岐点係数2）】<br>・索敵スコア **70以上** で **O（ボス）** 確定進行<br>・索敵スコア70未満 → I（航空戦・逸れ） または H（戦闘回避・逸れ） |

---

### 3. おすすめ編成例
#### ◆ 五航戦史実最短ルート編成（推奨）
*   **編成**: 翔鶴 ＋ 瑞鶴 ＋ 高速戦艦2 ＋ 駆逐2
*   **ルート**: B → C（空襲） → D（通常/対潜） → F（通常） → O（ボス）（道中3戦）
*   **速力**: **高速統一**（低速戦艦を起用する場合はタービン等での高速化が必須）
*   **制空値**: 270以上（道中の空襲およびボスの空母棲鬼パターンでの制空優勢を狙う）
*   **索敵スコア**: 分岐点係数2で **70以上**
*   **特徴**: 五航戦のルート固定により、Dマスを経由しつつ戦艦を2隻編成できます。道中・ボスともに最も安定する編成です。駆逐艦の1隻には対空カットインを装備させて空襲戦対策をしてください。

#### ◆ 重量・高速不問編成（五航戦未所持向け）
