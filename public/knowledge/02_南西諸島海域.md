## 本海域の特徴

-   「elite」と呼ばれる中級レベルの敵艦や、2-3,2-4,2-5では「flagship」と呼ばれる上級レベルの敵艦が登場します。
    -   flagshipは非常に強力で、改造して近代化改修を十分行った艦でなければ太刀打ちすることは難しいです。
        -   近代化改修では、昼戦の攻防に関わる火力と装甲をまず重点的に強化しましょう。
    -   基本戦略としては「**制空権を取って殴る**」、これにつきます。
        -   航空戦で**航空優勢**以上を取ることで、戦艦・重巡等は「弾着観測射撃」、空母は「戦爆連合カットイン攻撃」を発動可能です。
        -   「艦載機熟練度」の高い艦戦を用意することで航空優勢を取りやすくなり、攻略を優位に進められるでしょう。
-   第二期からの大きな相違点として、この2-X以降、各プレイヤーの司令部レベルにより、敵編成が大きく変化していくことが確認されました。  
    （第一期でもEO海域等で実装されていたシステムですが、通常海域にも導入されることになった模様）
    -   司令部レベルが低いうちは、旧海域とほぼ変わらない（または全くの同一）編成ですが、司令部レベルが高くなってくると、敵編成が道中、ボスともに強化されます。  
        司令部レベルが低い提督においては、3-X以降においても、項目上段のリンク先から、旧海域の敵編成を参考にすると良いかもしれません。  
        （現在、司令部レベル100以上でほぼ確実に編成強化、80～90前後が強化ボーダーでは？と推測されています）
    -   これに伴い、強化後編成では、道中マスから後期型駆逐艦も登場するようになりました。後期型駆逐艦はflagship駆逐艦並みの性能を持ち、対潜攻撃も強力な為、脅威的存在となります。  
        そして、それ以上に脅威なのが圧倒的な回避性能。カットインや連撃などが乗っていない単発の砲撃は、まず当たらないと思った方がよいでしょう。
-   4つの通常作戦海域と、1つの追加作戦海域からなります。
    -   第一期と比較した場合、羅針盤制御が容易になっている一方、大型艦中心ではボスルートから逸れたり、敵編成がかなり強化されているなど、  
        正面からのゴリ押しが利き辛く、周回に対する被害が大きくなりがちな傾向となっています。
    -   また羅針盤制御について、当海域の特徴として2-1～2-3のボスマスに向かうためには水母が重要な役割を果たします。
    -   特に任務挑戦時などで効率良い周回を目指す際に、編成に組み込む機会が増えるでしょう。  
        千歳・千代田以外は入手性に難があるため、瑞穂などを入手するまでは軽空母艦種とは別に、千歳甲・千代田甲のいずれかを確保しておくと役に立ちます。
-   上述の通り、後期型駆逐艦が出現したことによって、潜水艦が中・大破しやすくなりました。  
    一部任務には、依然として潜水艦クルージングが有効なケースもありますが、戦闘終了後の自軍被害の確認はしっかりと行いましょう。
-   任務で出撃する際には「沖ノ島**海域**(2-4)」と「沖ノ島**沖**(2-5)」が非常に間違えやすいので注意しましょう。

## 2-1. 南西諸島近海

### 1. 海域概要・特徴
南西諸島海域の最初の海域です。敵空母や戦艦が本格的に出現し始めます。
*   **作戦名**: 南西諸島哨戒
*   **クリア条件**: ボス（Hマス）でB勝利以上
*   **主な出現アイテム**: 鋼材（Bマス）、高速建造材（Eマス）
*   **特徴**: 敵空母・戦艦が出現し始めるため、十分な制空値の確保が重要になります。編成の艦種・隻数および速力によって、最短1戦でボスマスへ到達するルート（C → E → H）を固定可能です。

---

### 2. ルート分岐法則
空母系の数や特定の軽量編成、高速統一によってボス（H）へ固定ルートを構成できます。

| 分岐点 | 進行マス | 条件 |
| --- | --- | --- |
| **スタート / C** | **B**（鋼材） / **D** / **E**（高速建造材） | ・空母系3隻以上、または航空戦艦2隻以上、または「補給艦1隻以上かつ潜水艦0隻」で **B** 確定<br>・航空戦艦1隻編成の場合：(水母/潜母)1隻以上でB寄りランダム、それ以外はBまたはDランダム<br>・航空戦艦0隻かつ(水母/潜母)1隻以上で **E** 確定<br>・それ以外はDまたはEのランダム（各50%） |
| **D** | **H**（ボス） | 固定 |
| **E**（高速建造材） | **D** / **F** / **H**（ボス） | ・(戦艦級+空母系)5隻以上で **F** 確定<br>・【6隻編成時】(駆逐+海防)6隻、または軽巡1隻＋(駆逐+海防)5隻、または【高速統一】かつ軽巡1隻＋駆逐4隻で **H** 確定（それ以外はD）<br>・【5隻以下編成時】(駆逐+海防)5隻、または軽巡1隻＋(駆逐+海防)4隻、または【高速統一】かつ軽巡1隻＋駆逐3隻で **H** 確定（それ以外はD寄りランダム） |
| **F** | **G**（戦闘回避） / **H**（ボス） | ・(戦艦級+空母系)5隻以上で **G** 確定<br>・駆逐3隻以上、または軽巡1隻以上＋駆逐2隻以上で **H** 確定<br>・それ以外はH寄りランダム（60%でH） |

### 3. おすすめ編成例
#### ◆ 最短道中1戦ルート編成（周回・任務向け）
*   **編成**: 軽巡 1隻 ＋ 駆逐 4隻 ＋ 水上機母艦 1隻（または潜水母艦 1隻）
*   **ルート**: C → E → H (道中1戦のみ)
*   **速力**: **高速統一**（低速水母を採用する場合はタービン＋強化型コブラによる高速化が必要）
*   **制空値**: 81以上（ボスマス航空優勢）または162以上（制空権確保）
*   **陣形**: すべて単縦陣
*   **特徴**: 高速統一かつ軽量編成にすることで、戦闘がCマスの1回のみでボスに到達します。水母の開幕爆撃と甲標的による先制雷撃で敵を減らして進む、バケツ消費を抑えやすい周回用編成です。

#### ◆ 3戦安定・初攻略編成（戦艦・空母運用）
*   **編成**: 正規空母 1隻 ＋ 軽空母 1隻 ＋ 重巡 2隻 ＋ 軽巡 2隻（※戦艦2隻まで交代可、航戦は不可）
*   **ルート**: C → D → H または C → E → D → H (道中2戦)
*   **制空値**: 81〜162以上（ボスマス優勢〜確保）
*   **陣形**: すべて単縦陣
*   **特徴**: 空母2隻の開幕航空攻撃で道中の敵を削り、ボスマスでは弾着観測射撃による火力を叩き込む、安定感の高い攻略編成です。

#### ◆ 潜水艦6隻周回編成（コスト節約・デイリー消化用）
*   **編成**: 潜水艦/潜水空母 6隻
*   **ルート**: C → E → D → H（道中2戦）
*   **特徴**: 燃費が非常に良く、ボスマスでの南西デイリー任務やい号作戦（敵空母撃破）の同時消化に向いています。後期型駆逐艦などの対潜攻撃による被弾大破に注意してください。

### 6. 特記事項・ギミック
*   **司令部レベルによる敵難易度変化**:

*   **高速建造材（バーナー）の獲得**:
    *   Eマスを踏むことで高速建造材（バーナー）を確実に獲得できます。昨今では特定艦（吹雪改三など）の改装用やイベントの友軍艦隊要請で大量消費するため、水母1を含む編成で「C→E→撤退」や「C→E→D→撤退」を繰り返すバーナー回収周回が行われることがあります。

## 2-2. バシー海峡

### 1. 海域概要・特徴
南西諸島海域の2番目の海域です。
*   **作戦名**: 柳作戦
*   **クリア条件**: ボス（Kマス）でB勝利以上
*   **主な出現アイテム**: ボーキサイト（Aマス、Fマス）、高速建造材（Jマス）
*   **特徴**: 敵に戦艦ル級が出現し、司令部Lv上昇に伴いelite化するため道中・ボスマスともに大破リスクが高まります。一方で、水上機母艦（水母）や海防艦を適切に編成することで、最短道中1戦のみでボスマス（K）へ固定することができます。

---

### 2. ルート分岐法則
水母や潜水母艦の編成がスタートからの羅針盤固定に必須となります。

| 分岐点 | 進行マス | 条件 |
| --- | --- | --- |
| **スタート / C** | **B**（輸送艦/ボーキ） / **D** / **E**（通常） | ・空母系3隻以上、または航空戦艦2隻以上、または「補給艦1隻以上かつ潜水艦0隻」で **B** 確定（バシクルルート）<br>・航空戦艦1隻編成の場合：(水母/潜母)1隻以上でB寄り（70%）ランダム、それ以外はB/Dランダム<br>・航空戦艦0隻かつ(水母/潜母)1隻以上で **E** 確定（ボスルート）<br>・それ以外はDまたはEのランダム（各50%） |
| **E**（通常） | **F**（ボーキ） / **G**（通常戦） / **K**（ボス） | ・(戦艦級+空母系)4隻以上で **G** 確定<br>・海防艦2隻以上で **F** 確定<br>・(戦艦級+空母系)3隻でG寄り（70%）、2隻でG/Kランダム（50%）、1隻でK寄り（70%）ランダム<br>・(戦艦級+空母系)0隻かつ【高速統一】かつ軽巡1隻以上かつ(駆逐+海防)2隻以上で **K** 確定<br>・(駆逐+海防)3隻以上かつ潜水母艦を含むと **F** 確定 |
| **G**（通常戦） | **H**（戦闘回避） / **K**（ボス） | ・空母系を含む、または駆逐0隻で **H** 確定<br>・それ以外は基本K寄り（70%）ランダム |
| **H** | **I** / **J**（高速建造材） / **K**（ボス） | ・(空母系+航巡+水母)1隻以上で **K** 確定<br>・(駆逐+海防)2隻以上でJまたはKへ進行 |

---

### 3. おすすめ編成例
#### ◆ 道中1戦ボス確定編成（最安定・大型艦入り）
*   **編成**: 正規空母（または軽空母） 2隻 ＋ 軽巡 1隻 ＋ 水上機母艦 1隻 ＋ 海防艦 2隻
*   **ルート**: C → E → F（ボーキ） → H → K (道中1戦ボス確定・Fマスでボーキ回収可能)
*   **制空値**: 81以上（優勢）または162以上（確保）
*   **陣形**: Eマス（単単縦陣）／ Kマス（単縦陣）
*   **特徴**: 空母2隻の開幕航空火力で敵を制圧しつつ、海防艦2隻と水母1隻を入れることで道中1戦ボス確定となります。Fマスでボーキサイトも回収できる非常に優れた周回・攻略編成です。

#### ◆ 道中1戦ボス確定編成（高速統一水雷・巡洋艦主軸）
*   **編成**: 重巡（または航巡） 2隻 ＋ 軽巡 1隻 ＋ 駆逐 2隻 ＋ 水上機母艦 1隻
*   **ルート**: C → E → K (道中1戦ボス確定)
*   **速力**: **高速統一**
*   **特徴**: 海防艦を使わずに重巡/航巡の火力でボスを叩く編成です。低速の水母を採用する場合は高速化が必須となります。

#### ◆ 補給艦撃破＆ボーキサイト回収（バシクル周回 / 1戦撤退）
*   **編成**: 軽空母 3隻 ＋ 駆逐 3隻（または軽空母3 ＋ 駆逐2 ＋ 潜水艦1）
*   **ルート**: C → B（通常/輸送艦） → A（ボーキ獲得） → 撤退
*   **装備**: 駆逐艦にはドラム缶や大発動艇をフル装備
*   **特徴**: 空母系3隻を編成することでスタートからBマス（輸送ワ級が確実に出現するマス）へ固定されます。Bマスでデイリー/ウィークリーの輸送艦撃破任務を消化後、Aマスでボーキサイトを回収して撤退します（バシクル）。潜水艦をデコイとして1隻入れておくと駆逐艦の大破を防ぎやすくなります。

## 【Extra Operation】2-5. 沖ノ島沖

### 1. 海域概要・特徴
南西諸島海域のExtra Operation（EO）海域です。
*   **作戦名**: 沖ノ島沖戦闘哨戒
*   **クリア条件**: ボス（Oマス）の旗艦を4回撃破でクリア（戦力ゲージは毎月1日の0:00に自動リセット）。
*   **海域クリア報酬**: 勲章×1、特別戦果+100
*   **特徴**: ボスへのルート固定およびボス到達のために、十分な **索敵値（索敵スコア）** の確保が極めて重要です。重量編成で挑む「北ルート」と、軽量高速編成で挑む「南ルート」の2つの主要攻略ルートが存在します。

### 3. おすすめ編成例
#### ◆ 南3戦最短ルート編成（低燃費・周回推奨）
*   **編成**: 軽空母 1隻 ＋ 正規空母（または軽空母） 1隻 ＋ 軽巡 1隻 ＋ 駆逐 3隻
*   **ルート**: C → E → I → O (道中2戦・最短固定)
*   **速力**: **高速統一**（低速艦を編成する場合はタービン装備等で高速化が必須）
*   **制空値**: 100以上（全マスで制空権確保）
*   **索敵スコア**: 分岐点係数1で **34以上**（電探や彩雲、水偵を十分に装備させること）
*   **特徴**: 最も消費資源が少なく、周回に適した編成です。空母2隻の開幕航空戦で道中の敵を殲滅し、被弾を減らしてボスを叩きます。

#### ◆ 北4戦重量編成（妙高型任務「第五戦隊」向け）
*   **編成**: 重巡 2隻 ＋ 航巡 2隻 ＋ 航空戦艦 2隻（低速艦を含むこと）
*   **ルート**: B → F → J → O (道中3戦・うずしお回避)
*   **装備**: 航巡2隻にそれぞれ「ドラム缶」を1個以上装備させること（B固定用）
*   **索敵スコア**: 分岐点係数1で **49以上**
*   **特徴**: 戦艦と重巡を多数投入できるため、昼戦の火力と耐久力に優れます。Jマスのル級flagshipの砲撃に注意してください。

#### ◆ 水上反撃部隊編成（マンスリー任務「水上反撃部隊」突入せよ！向け）
*   **編成**: 駆逐 1隻（**必ず旗艦**） ＋ 重巡 1隻 ＋ 軽巡 1隻 ＋ 駆逐 3隻
*   **ルート**: B → F → E → I → O (道中3戦・高速統一)
*   **索敵スコア**: 分岐点係数1で **34以上**（駆逐艦に電探、重巡/軽巡に水偵や水戦を多めに搭載すること）
*   **特徴**: マンスリー難関任務の指定編成です。航巡が編成不可のため、ボスの空母ヲ級入り編成に対して制空権を取るために、重巡（Zara dueなど）や軽巡（由良改二、多摩改二、Gotland等）に水戦を装備させて制空拮抗〜優勢を狙い、敵戦艦の弾着観測射撃を封じるのがセオリーです。
