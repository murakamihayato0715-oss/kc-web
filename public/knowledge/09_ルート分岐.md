# ルート分岐の基本法則 (Route Branching Rules)

海域の各マスを移動する際、艦隊の編成（艦種、隻数、速力、特定の装備など）によって進むルートが分岐する。

---

## 1. 主要な艦種区分と略記
ルート分岐条件で頻出する艦種グループおよび個別の略記は以下の通り。

| 艦種名 | 略記 | グループ定義・備考 |
| --- | --- | --- |
| 海防艦 | 海防 / DE | 主に輸送護衛や対潜ルートで駆逐艦の代用・補助として使われる。 |
| 駆逐艦 | 駆逐 / DD | ほぼ全ての海域でルート固定に必須となる最重要艦種。 |
| 軽巡洋艦 | 軽巡 / CL | 水雷戦隊の旗艦やルート固定で広く使用される。 |
| 重雷装巡洋艦 | 雷巡 / CLT | 先制雷撃が強力だが、ルート分岐上は「雷巡」として弾かれることが多い。 |
| 重巡洋艦 / 航空巡洋艦 | 重巡 / CA, 航巡 / CAV | 主力水上打撃部隊。航巡はドラム缶や水爆・水戦を搭載可能。 |
| 戦艦級 | BB / FBB / BBV | 高速戦艦(FBB), 航空戦艦(BBV)を含む。火力が高い反面、多すぎると逸れる原因になる。 |
| 空母系 | CV / CVL / CVB | 正規空母(CV), 装甲空母(CVB), 軽空母(CVL)を含む。制空の要だが、同様に隻数制限が厳しい。 |
| 水上機母艦 | 水母 / AV | 甲標的による開幕雷撃や水爆・水戦搭載が可能。ルート上で特殊な固定枠を持つ。 |
| 潜水艦 / 潜水空母 | 潜水 / SS, 潜母 / SSV | デコイ（囮）として有効。一部海域では潜水艦のみの最短ルート等が存在。 |

---

## 2. 速力（Speed）による分岐
艦隊全体の速力によってルートが変わる海域が非常に多い。
- **速力区分**: `超高速` > `高速+` > `高速` > `低速`
- **艦隊速力ルール**:
  - 艦隊の速力は **「艦隊の中で最も遅い艦の速力」** が基準となる（例：高速艦5隻に低速艦1隻が混ざると、艦隊全体は「低速」扱いになる）。
- **速力の引き上げ（速力シナジー）**:
  - 本来低速の艦であっても、タービン（改良型艦本式タービン）とボイラー（強化型艦本式缶・新型高温高圧缶）を装備スロット（または補強増設スロット）に組み合わせることで、速力を「高速」や「高速+」に引き上げ、ルート逸れを防ぐことができる。

---

## 3. その他のルート分岐要素
- **索敵値**:
  - ボスマス前などで一定以上の索敵値（分岐点係数を用いた計算）が必要となる。索敵値が足りないと逸れる（「逸れ」や「お仕置きマス」へ行く）。水偵、電探、艦偵の装備が重要。
- **ドラム缶（簡易輸送部隊用） / 大発動艇**:
  - 主に輸送マップ（TPゲージ）や一部の特定マップで、一定個数以上の装備がルート固定条件になる。
- **電探搭載艦数**:
  - 渦潮マスでの被害軽減や、一部の電探ナビゲーションルート（特定の索敵やルート固定）で必要となる。
