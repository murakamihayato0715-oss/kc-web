## 本海域の特徴

-   ここからはflagshipが当たり前のように出てきます。改造や近代化改修を既に済ませ、ある程度強力な装備を確保していることが前提の難易度と言えます。
-   鎮守府海域\*1や南西諸島海域と同じく、全マップにボスへのルート固定編成が存在します。
    -   駆逐艦メインの編成でなければボスに着けない3-2、確定で渦潮を踏む3-3、重量編成だと羅針盤に振り回される3-4など、全体的に癖が強いです。
-   ドロップする艦娘が今まで以上に豊富になります。
    -   慣れてきたなら、周回が比較的楽な海域を回ってみるのも良いでしょう。
-   2014/9/12のアップデートで3-5(EO)が実装。編成の制約がきつめで、ボスまでの道中が厳しいのが特徴です。
    -   戦艦・空母の有無で敵編成が大きく変わるほか、編成次第でルートの安定度が大きく変わります。どんな艦種を組み合わせて攻略するか、自分のスタイルに合わせて探ってみましょう。
-   2017/10/18のアップデートでマップ進撃中のBGMが「北東方面艦隊の集結」に変更されました。

## 3-1. モーレイ海

### 1. 海域概要・特徴
北方海域の最初の海域。3-1からはflagship級の敵が多く出現するようになります。
*   **作戦名**: モーレイ海哨戒
*   **難度**: ☆☆☆☆
*   **クリア条件**: ボスB勝利以上
*   **主な入手アイテム**: 弾薬
*   **特徴**: 敵は水上艦中心で航空戦力は貧弱です。制空値約90で全マス制空権確保が可能。一方、**重巡リ級flagship**が多く、約80の火力は駆逐に大破を与えやすく注意が必要です。道中2戦の最短ルートが推奨されますが、超重量編成（戦艦・空母のみ）はボス前でEマスに逸れるため編成に注意が必要です。

---

### 2. ルート分岐法則
艦隊の戦艦・空母の数と駆逐の数でルートが決まります。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **C** | (駆逐+海防)1以下 → D<br>(航戦+水母+補給+潜母+軽巡)3以上 → 50%でB<br>(戦艦+空母)3以上 → D<br>(水母+補給+潜母)0 → F<br>(駆逐+海防)2以上 → F<br>その他 → 50%でB / 50%でF |
| **D** | (戦艦+空母)5以上 → E（逸れ）<br>潜水6 → E<br>潜母1かつ潜水5 → G<br>その他 → F |
| **F** | G固定 |

**ルート固定条件**

| 編成 | ルート | 条件 |
| --- | --- | --- |
| (戦艦+空母)2・(駆逐+海防)2・自由2 | CFG（道中2戦・推奨） | (航戦+軽巡+水母+補給)2以下に注意 |

## 3-2. キス島沖

### 1. 海域概要・特徴
北方海域の第2海域。高難度と呼ばれる特殊な海域で、ボスへの到達編成が極めて限定されています。
*   **作戦名**: キス島撤退作戦
*   **難度**: ☆☆☆☆☆
*   **クリア条件**: ボスB勝利以上
*   **主な入手アイテム**: 弾薬、家具箱（小）
*   **出撃解放条件**: 3-1クリア **かつ** 1-5のゲージ破壊完了
*   **特徴**: 軽巡・駆逐・補給艦のみがボスに到達できる特殊海域です。戦艦・空母・重巡・雷巡を1隻でも編成するとボスに到達不可となります。また **5隻以下ではボスに到達不可**（CGHI固定）のため、**6隻出撃が必須**です。

**ボス到達可能編成（いずれか）**:
*   軽巡1 ＋ 駆逐5
*   軽巡1 ＋ 駆逐4 ＋ 補給艦1
*   駆逐6（初手Aスタートになる）

---

### 2. ルート分岐法則
速力（高速+統一/最速統一）と電探装備数がルートに大きく影響します。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **始点** | (戦艦+空母)1以上 → C<br>駆逐6 → C<br>軽巡1かつ駆逐4以上 → C（軽巡0や2は不可）<br>その他 → A（ボス不達） |
| **C** | 駆逐3以下 → A（ボス不達）<br>(戦艦+空母)2以上 → A（ボス不達）<br>潜水1以上 → ランダムでAかG<br>低速含む → G<br>電探装備艦0 → G<br>(軽巡+駆逐+補給)5以下 → G<br>最速統一かつ電探4以上 → E（Hを確定回避）<br>高速+統一 → 40%でE / 60%でG<br>補給1以上 → 45%でE / 55%でG<br>その他 → G |
| **E** | 高速+統一以上 → F（ボスへ）<br>その他 → 約17%でD / 83%でF |

### 3. おすすめ編成例
#### ◆ 初攻略・通常編成（軽巡1駆逐5）
*   **編成**: 軽巡1 ＋ 駆逐5
*   **ルート**: C → G（渦潮） → (H) → F → L（ボス）（道中2〜3戦）
*   **電探**: 1隻以上に電探を装備（G→Fの確率が35%向上）
*   **陣形**: 全戦単縦陣
*   **特徴**: Gマスの渦潮による燃料喪失あり。Hマス（戦艦ル級elite登場）はランダム経由。ダメコンの使用も有効。まずは改装・近代化改修を最大まで施した軽巡・駆逐5隻を揃えることを優先しましょう。

#### ◆ 高速+統一（Hマス確定回避・安定周回向け）
*   **編成**: 軽巡1 ＋ 駆逐5（全艦高速+以上）
*   **ルート**: C → E → F → L（ボス）またはC → G → F → L（渦潮60%）（道中2戦）
*   **必要装備**: 新型高温高圧缶＋改良型艦本式タービン（各艦1セット）＋電探1以上
*   **特徴**: Hマス（戦艦ル級elite）を確率的に回避可能。補強増設があれば主砲＋缶タービン＋電探の両立が容易に。初攻略時は無理に揃える必要はなし。
