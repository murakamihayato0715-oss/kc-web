## 本海域の特徴

-   2014/09/26実装。同日に6-1、6-2が、2015/06/26に6-3が、2016/03/11に6-4が、2016/10/05に6-5が実装。
    -   南方海域クリア後に出撃可能になることもあり、出現する敵は最上位クラスの「改flagship」や「鬼、姫」など非常に強力です。
        -   南方海域同様**軽巡ツ級**には注意。回避力が非常に高く、その上対空や雷装も高いため攻撃機全滅や砲雷撃を高確率で回避、雷撃ターンで大破をもらうということも。  
            ちなみにノーマルとeliteがいますが、違いは攻撃・耐久面だけで**防空能力はほとんど変わりません。**撃墜数は固定9機、割合23%前後です。
-   敵編成がきわめて強力な一方で、こちら側の編成は全体的に厳しく制限されています。
    -   6-1のボス攻略には**潜水艦3隻が必須**です。
    -   6-3は水雷戦隊+水母で航空偵察を行うマップです。**水母、軽巡、練巡、駆逐のみ出撃可**となっています。
    -   6-4のボス艦隊は**陸上型深海棲艦**を多数含みます。また、編成によってスタート地点が変わり、ルート条件も複雑です。
    -   6-5のボス艦隊はこちらが通常艦隊なのに対し、**深海棲艦初の連合艦隊**です。
-   どのマップも攻略には非常に高い練度と充実した装備、そしてかなりの試行回数を必要とするため、気長にやるのが良いでしょう。
    -   全マップにルート固定条件があるため、ルート固定はある程度可能。また6-4以外はボス前のルート分岐で索敵値を要求されます。
-   2016/10/05のアップデートで、6-4及び同日追加の6-5に基地航空隊を出すことが可能になりました。詳細は個別ページ参照。
-   難易度に応じるようにドロップは豪華なので、掘るにはお勧めです。
    -   6-1では伊401、まるゆなど多くの潜水艦がドロップで入手でき、極低確率ですが大鯨もドロップします。
    -   6-2ではドロップ限定の酒匂や、大型建造のみで建造可能な矢矧がドロップ可能なので、未所持提督は是非狙いたいところ。また明石やまるゆも手に入ります。
    -   6-3ではボスS勝利で春雨、雲龍がドロップします。
    -   6-4では秋津洲、速吸のほか、長らくイベント海域でしか出なかった時津風がドロップします。
    -   6-5では高波、海風、沖波、練習巡洋艦鹿島など、通常海域では初のドロップが確認されています。
-   各海域がリニューアルされるなか、中部海域は全域が第一期のままになっています。
    -   ただし攻略は残念ながらやり直し。しかし対潜戦や開幕夜戦での燃料弾薬消費が減ったりと昔より緩和されている部分もあります。
    -   他の海域の難易度が相当上昇するなか、全域据え置きと南方海域までより簡単という人もいるかもしれません。
        -   特に6-5の難易度に関しては戦果250と全EOで最大の入手量ながら難易度表記は☆7と第一期の☆13から半分近くに低下しています。

## 6-1. 中部海域哨戒線

### 1. 海域概要・特徴
中部海域の最初のマップです。クリア条件はボス（Kマス）B勝利以上です。
*   **作戦名**: 潜水艦作戦
*   **難度**: ☆☆☆☆☆☆☆☆
*   **クリア条件**: ボスB勝利以上（戦力ゲージなし・恒久クリア）
*   **特徴**:
    *   ボスへ到達するために**潜水艦（または潜水空母）が最低3隻必要**な海域です。
    *   ほぼすべての戦闘マスに、ソナー・爆雷を装備した敵駆逐艦・軽巡洋艦が多数出現するため、潜水艦の大破撤退率が非常に高いのが特徴です。
    *   水上艦主体の編成で道中を安定させるか、潜水母艦を組み込んでルートを完全固定するかが攻略の鍵となります。

---

### 2. ルート分岐法則
艦隊内の潜水艦・潜水母艦の数、戦艦や空母などの大型艦の数、および索敵スコア（分岐点係数4）によってルートが決定されます。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **スタート** | ・(戦艦級+空母系+重巡級)3隻以上、または戦艦級2隻以上 → **B**（戦闘回避）<br>・潜水艦3隻以上かつ潜水艦のみの艦隊 → **A**（戦闘回避）<br>・潜水母艦1隻 かつ 潜水艦3隻以上 かつ それ以外の艦種なし → **A**（戦闘回避）<br>・潜水母艦1隻 かつ 潜水艦3隻 かつ 駆逐2隻 → **A**（戦闘回避）<br>・潜水母艦1隻 かつ 潜水艦4隻 かつ (軽巡or駆逐)1隻 → **A**（戦闘回避）<br>・(軽巡+駆逐)0隻 → **B**（戦闘回避）<br>・それ以外は **C**（通常） |
| **A** | ・潜水母艦を含む → **F**（通常）<br>・それ以外は **D**（通常） |
| **C** | **F** 確定 |
| **D / F** | **G**（戦闘回避） 確定 |
| **G** | ・潜水艦2隻以下、または(戦艦級+空母系+重巡級)2隻以上 → **I**（逸れ・空母棲鬼）<br>・潜水母艦なしの場合、索敵を満たしていても約15%の確率で無条件で I へ逸れる<br>・【索敵スコアチェック（分岐点係数4）】<br>・索敵スコア **16以上**（かつ潜水母艦あり）で **H** 確定進行、12未満でI逸れ |
| **H** | ・【索敵スコアチェック（分岐点係数4）】<br>・**潜水母艦を含む場合**: 索敵スコア **25以上** で **K（ボス）** 確定、20未満でE（逸れ）<br>・**潜水母艦を含まない場合**: 索敵スコア **36以上** で **K（ボス）** 確定、20未満でE逸れ |

### 3. おすすめ編成例
#### ◆ 水上混成編成（マンスリー任務・S勝利重視）
*   **編成**: 正規空母1 ＋ 軽巡1 ＋ 雷巡（または軽巡/水母）1 ＋ 潜水艦3
*   **ルート**: C → F（通常） → G → H（通常） → K（ボス）（道中3戦・Gマスで約15%のランダムI逸れあり）
*   **制空値**: 130以上（道中確保、ボス航空優勢）
*   **索敵スコア**: 分岐点係数4で **36以上**（空母に彩雲、軽巡に水偵を装備）
*   **特徴**: 空母の開幕攻撃と水上艦の昼砲撃で敵の対潜艦を先制撃破し、潜水艦の被弾を最小限に抑えます。ボス完全固定はできませんが、最も道中が安定しボスS勝利を狙いやすい編成です。矢矧改二乙などの高火力軽巡を旗艦に置くのが推奨されます。

#### ◆ 潜水母艦ルート固定編成（周回・資材節約向け）
*   **編成**: 潜水母艦1 ＋ 軽巡1 ＋ 潜水艦4
*   **ルート**: A → F（通常） → G → H（通常） → K（ボス）（道中3戦・ボス完全固定）
*   **速力**: 高速統一不要（低速の潜水母艦をそのまま起用可能）
*   **索敵スコア**: 分岐点係数4で **25以上**（潜水母艦に高索敵水偵を装備）
*   **特徴**: 潜水母艦を編成することでGマスでのランダム逸れを排除し、ボスへルート固定します。潜水母艦の戦闘力が低いため道中撤退率は上がりますが、S勝利を要求されないクォータリー任務「Z作戦（A勝利以上）」や資材を抑えたい場合に有効です。

### 6. 特記事項・ギミック
*   **対潜攻撃への対策**: 敵の駆逐後期型や軽巡flagshipは非常に高い対潜攻撃力を持っているため、こちらの陣形は開幕雷撃で敵を即座に沈める「単縦陣」一択になります。
*   **Iマスのまるゆ/伊401掘り**: 潜水艦を一切含まない水上艦隊（例：空母2 ＋ 雷巡3 ＋ 軽巡1）で出撃すると、スタートからC→F→G→Iと完全にIマスへルート固定され、まるゆや伊401の掘り周回が可能です。

## 6-2. MS諸島沖

### 1. 海域概要・特徴
道中に強力な戦艦や空母が多数出現する、中〜上級者向けの海域です。クリア条件はボス（Kマス）旗艦の3回撃沈です。
*   **作戦名**: MS諸島防衛戦
*   **難度**: ☆☆☆☆☆☆☆☆☆
*   **クリア条件**: ボス旗艦3回撃破（戦力ゲージクリア後は復活なし）
*   **特徴**:
    *   道中のほぼ全戦闘マスに空母または戦艦が出現し、かつ対空砲火の強力な「軽巡ツ級」が確定出現するため、艦載機の撃墜対策や対空カットイン艦の編成が非常に重要になります。
    *   通常は、うずしおを回避する「南3戦最短ルート（CEJK）」で攻略するのが一般的です。通常海域で唯一、軽巡「酒匂」がボスドロップします。

---

### 2. ルート分岐法則
艦隊の重量（戦艦・空母の数）や、軽巡・駆逐の隻数、速力および索敵スコア（分岐点係数3）によってルートが決定されます。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **スタート** | ・(軽巡+駆逐)4隻以上 → **B**（通常/対潜）<br>・(航空戦艦+航巡+水母+揚陸艦)1隻以下 かつ 潜水艦4隻以下の場合：<br>　・(戦艦級+空母系)5隻以上 → **B**<br>　・(戦艦級+空母系)4隻 → B寄り(65%)のランダム<br>・それ以外は **C**（通常） |
| **B** | ・(軽巡+駆逐)5隻以上 → **D**（うずしお）<br>・空母系2隻以下 かつ 戦艦級0隻 → C寄り(70%)のランダム<br>・それ以外は **C**（通常） |
| **C** | ・潜水艦6隻、または(戦艦級+空母系)5隻以上、または(戦艦+空母+重巡)6隻 → **A**（戦闘回避・逸れ）<br>・(戦艦級+空母系)2隻以下 → **E**（通常）<br>・それ以外は **D**（うずしお） |

| **E** | ・戦艦級2隻以上、空母系2隻以上、または駆逐1隻以下 → **F**（通常/空母）<br>・【索敵スコアチェック（分岐点係数3）】<br>　・索敵スコア **50以上** で **J**（通常/戦艦） 確定進行、43未満でI（通常/戦艦） |
| **F** | **I**（通常/戦艦） 確定 |
| **H** | ・【索敵スコアチェック（分岐点係数3）】<br>　・索敵スコア **32以上** で **K（ボス）** 確定、32未満でG（逸れ） |
| **I** | ・潜水艦4隻以上 → G（逸れ）<br>・【索敵スコアチェック（分岐点係数3）】<br>　・索敵スコア **40以上** で **K（ボス）** 確定、35未満でG（逸れ） |
| **J** | **K（ボス）** 確定 |

### 3. おすすめ編成例
#### ◆ 南3戦最短ルート編成（推奨・初攻略）
*   **編成**: 戦艦1 ＋ 正規空母1 ＋ 駆逐2 ＋ 雷巡1 ＋ 航巡1
*   **ルート**: C（通常） → E → J（通常/戦艦） → K（ボス）（道中3戦・うずしお回避）
*   **速力**: 高速統一不要（低速戦艦をそのまま起用可能）
*   **制空値**: 160以上（Jマスのヲ級改やボスのヲ級に対して航空優勢を確保）
*   **索敵スコア**: 分岐点係数3で **50以上**
*   **特徴**: 最も主流となる攻略編成です。雷巡の開幕雷撃で敵を減らし、航巡の水戦等で制空を稼ぎます。駆逐艦の1隻には対空カットインを装備させて空襲被害を抑えましょう。

#### ◆ 南4戦重量編成（駆逐艦なし・道中安定重視）
*   **編成**: 正規空母2 ＋ 重巡（または航巡）2 ＋ 雷巡1 ＋ 軽巡1
*   **ルート**: C → E → F（通常/空母） → I（通常/戦艦） → K（ボス）（道中4戦・うずしお回避）
*   **制空値**: 260以上（Fマスのヲ級flagship×2に対して航空優勢を狙う）
*   **索敵スコア**: 分岐点係数3で **40以上**
*   **特徴**: 駆逐艦を一切編成せず、耐久・装甲が高い艦だけで構成された重量編成です。道中のFマスが追加されるためボーキサイトの消費は多くなりますが、道中撤退率は抑えられます。

## 6-5. KW環礁沖海域

### 1. 海域概要・特徴
敵艦隊が強力な連合艦隊（12隻）を編成しており、基地航空隊と本隊の制空権・対空防御の連携が重視されるExtra Operation（EO）海域です。
*   **作戦名**: 空母機動部隊迎撃戦
*   **難度**: ☆☆☆☆☆☆☆
*   **クリア条件**: ボス（Mマス）旗艦「空母棲姫」を6回撃破（戦力ゲージクリア後は復活なし）
*   **特徴**:
    *   ボスの主力艦隊は連合艦隊編成となっており、夜戦でボス旗艦（空母棲姫）にダメージを与えるには、昼戦の間に敵護衛部隊（第2艦隊）をほぼ全滅させるか、大破撤退に追い込む必要があります。
    *   スタート地点が2箇所あり、空母や雷巡を含むと「上スタート」、これらを編成しない場合は「下スタート」になります。通常攻略では道中戦が比較的安定する**下ルート（BFIJM）**が主力になります。
    *   **基地航空隊**が最大2部隊出撃可能です（ボスマスへの必要行動半径は**5**）。また、ゲージを2回削ると空襲が発生するため、防空も考慮する必要があります。

---

### 2. ルート分岐法則
艦隊内の戦艦・空母の隻数、雷巡の有無、および索敵スコア（分岐点係数3）によってルートが決定されます。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **スタート** | ・軽巡0隻、または空母系を含む、または雷巡を含む、または戦艦級4隻以上 → **A**（通常・上）<br>・それ以外は **B**（対潜・下） |
| **B** | ・戦艦級3隻、または駆逐1隻以下 → **C**（通常）<br>・それ以外は **F**（通常） |
| **C** | ・駆逐0隻、または雷巡2隻以上 → **E**（対潜）<br>・(戦艦級+空母系)4隻以上、または(戦艦級+空母系+重巡級)5隻以上 → **E**（対潜）<br>・それ以外は **D**（通常） |
| **E** | ・[空母系 ＋ 軽巡 ＋ 駆逐] をすべて含む → **H**（空襲）<br>・それ以外は **I**（通常） |
| **I** | ・軽巡0隻 → **H**（空襲）<br>・駆逐2隻以上 → **J**（夜戦）<br>・[戦艦0 かつ (空母+重巡級)4以下 かつ 空母2以下 かつ 重巡級4以下] → **J**（夜戦）<br>・それ以外は **H**（空襲） |
| **G** | ・【索敵スコアチェック（分岐点係数3）】<br>　・索敵スコア **50以上** で **M（ボス）** 確定、不足するとK（逸れ） |
| **J（夜戦）** | ・【索敵スコアチェック（分岐点係数3）】<br>　・索敵スコア **35以上** で **M（ボス）** 確定、不足するとL（逸れ） |

*   **下ルート【BFIJM】**: B（対潜） → F（通常） → I（通常/空母） → J（夜戦） → M（ボス）
*   **上ルート【ACDGM】**: A（通常） → C（通常/空母） → D（通常/空母） → G（空襲） → M（ボス）

### 3. おすすめ編成例
#### ◆ 下ルート：伊勢型改二制空編成（推奨・周回/ゲージ破壊向け）
*   **編成**: 航空戦艦2（伊勢改二＋日向改二） ＋ 軽巡1 ＋ 駆逐2 ＋ 重巡（または航巡）1
*   **ルート**: B → F → I → J（夜戦） → M（ボス）（道中4戦）
*   **陣形**: B:単横陣 / F:単縦陣 / I:単縦陣 / J:単縦陣 / ボス（M）:単縦陣
*   **制空値**: 250〜280程度（伊勢・日向の多スロットに艦戦を搭載し、ボス優勢を狙う）
*   **装備**:
    *   **航戦**: 主砲×2 ＋ 艦戦×2〜3 ＋ 徹甲弾 ＋ 水偵
    *   **軽巡・駆逐**: 1隻に対空カットイン（秋月型、Atlanta、矢矧など）を搭載。他は夜戦連撃装備。
*   **特徴**: 空母を含まず下ルートを通りながら、高い制空値と手数を維持できる最適編成です。Jマスの夜戦は単発主体の敵が多く、Iマスのヲ級改も防空体制が整っていれば比較的安全に突破できます。

#### ◆ 下ルート：ネルソンタッチ編成（突破率重視）
*   **編成**: 戦艦3（Nelson旗艦 ＋ 戦艦2） ＋ 軽巡1 ＋ 駆逐2
*   **ルート**: B → F → I → J（夜戦） → M（ボス）（道中4戦）
*   **特徴**: 複縦陣でボスマスまたは強敵マスに突入し、Nelsonの特殊攻撃「ネルソンタッチ」で随伴をなぎ払う編成です。戦艦が3隻必要なため、航戦を組み込んで制空値をカバーする必要があります。

#### ◆ 上ルート：空母機動編成（任務用・高火力）
*   **編成**: 戦艦2 ＋ 正規空母2 ＋ 軽巡1 ＋ 駆逐1
*   **ルート**: A → C → D → G（空襲） → M（ボス）（道中4戦）
*   **制空値**: 350〜400（Cマス、ボスマスで確実に優勢を確保）
*   **特徴**: 正規空母2隻の開幕航空攻撃で道中およびボスの随伴を削ります。Cマス（ヲ級改＋タ級）の砲撃戦が強烈なため、対空カットインによる無力化や戦艦の弾着観測射撃で早期撃破を狙います。

### 6. 特記事項・ギミック
*   **基地航空隊の運用**:
    *   2部隊をすべて**ボスマス（M）**へ集中派遣（行動半径5以上必須）するのが基本です。
    *   制空状態が「航空劣勢（第一波目安：制空値107以上）」になるよう調整（例：陸戦1陸攻3、または陸戦2陸攻2）し、ボスの制空値を削ると同時に敵随伴（第2艦隊）を撃破して本隊の航空優勢を支援します。
*   **基地防空ギミック**:
    *   ゲージを2回以上削ると、味方基地への空襲が1回発生するようになります。
    *   もし設営隊を使用して「第三航空隊」を開放している場合、これを「防空」設定にして高対爆の陸戦/局戦を配備しておくことで、基地へのダメージおよび資材損耗を無効化できます。
