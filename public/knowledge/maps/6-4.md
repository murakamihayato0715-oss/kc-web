## 6-4. 中部北海域ピーコック島沖

### 1. 海域概要・特徴
ボスおよび随伴の多くが陸上型深海棲艦で構成されており、各種対地装備や基地航空隊の的確な運用が攻略の鍵となる海域です。
*   **作戦名**: 離島再攻略作戦
*   **難度**: ☆☆☆☆☆☆☆☆☆
*   **クリア条件**: ボス（Nマス）旗艦の5回撃破（戦力ゲージクリア後は復活なし）
*   **特徴**:
    *   ボス（離島棲姫）および随伴（集積地棲姫、砲台小鬼）はすべて陸上型です。三式弾、大発動艇（八九式中戦車＆陸戦隊）、特二式内火艇、対地噴進砲などの**対地装備**の準備が必須となります。
    *   出撃する艦隊編成によって、開始地点が「左スタート」と「右スタート」に分かれます。道中の難易度の違いから、通常攻略や多くの任務では**左スタート**が推奨されます。
    *   **基地航空隊**が1部隊使用可能です（ボスマスへの必要行動半径は**5**、左スタート基準。噴式機がある場合は2）。

---

### 2. ルート分岐法則
艦隊内の戦艦・空母の隻数、特定の艦（秋津洲や長門型改二のペアなど）、および速力によってルートが決定されます。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **スタート** | ・揚陸艦1隻以上、空母系1隻以上、航巡3隻以上、または駆逐1隻以下 → **M**（対潜・右）<br>・(長門改二＋陸奥改二)のペアを除く戦艦級2隻以上 → **M**（対潜・右）<br>・低速艦を含む → **A**（通常・左）<br>・高速統一 かつ [軽巡(旗艦)1 ＋ 駆逐3以上] または [駆逐4以上] → **B**（通常・左・最短）<br>・それ以外は **A**（通常・左） |
| **A** | ・【秋津洲を含む場合】:<br>　・航巡1隻のみ、軽巡1隻以上、または駆逐3隻以上 → **D**（空襲）<br>　・それ以外は **E**（通常）<br>・【秋津洲を含まない場合】:<br>　・戦艦級を含む、または低速艦を含む → **E**（通常）<br>　・軽巡旗艦、または駆逐3隻以上 → **D**（空襲）<br>　・それ以外は **E**（通常） |
| **E** | ・秋津洲を含む、または如月を含む → **D**（空襲）<br>・低速艦を含む、航巡2隻以上、重巡2隻以上、または軽巡0隻 → **G**（空襲）<br>・それ以外は **D**（空襲） |
| **K（右）** | ・戦艦級2隻以上、(戦艦+重巡級)3隻以上、または駆逐1隻以下 → **H**（通常）<br>・それ以外は **J**（通常） |
| **J（右）** | ・軽巡0隻、駆逐1隻以下、または(戦艦+空母)3隻（※長門改二+陸奥改二の例外除く） → **L**（通常・逸れ）<br>・揚陸艦1隻以上 → **N（ボス）**（最短固定）<br>・低速艦を含み、かつ空母2隻以上 → **L**（通常・逸れ）<br>・それ以外は **I**（空襲） |

*   **左最短ルート【BDCFN】**: B（通常） → D（空襲） → C（通常/戦艦） → F（空襲） → N（ボス）
*   **左長陸奥秋津洲ルート【ADCFN】**: A（通常） → D（空襲） → C（通常/戦艦） → F（空襲） → N（ボス）
*   **右最短揚陸艦ルート【MKJN】**: M（対潜） → K（通常） → J（通常/空母） → N（ボス）

### 3. おすすめ編成例
#### ◆ 左最短ルート編成（高速統一・初攻略向け）
*   **編成**: 軽巡1（旗艦） ＋ 高速戦艦1 ＋ 航巡（または重巡/水母）1 ＋ 駆逐3
*   **ルート**: B → D（空襲） → C（通常/ル級） → F（空襲） → N（ボス）（道中4戦・高速統一）
*   **陣形**: 1戦目（B）:単縦陣 / 2戦目（D）:輪形陣 / 3戦目（C）:単縦陣 / 4戦目（F）:輪形陣 / ボス（N）:単縦陣
*   **制空値**: 35〜40程度（Cマスで航空拮抗にして敵の弾着観測射撃を防ぐ）
*   **装備**:
    *   **高速戦艦**: 主砲×2 ＋ 三式弾 ＋ 徹甲弾 ＋ 水偵（補強増設に三式弾を乗せて偵察機を両立）
    *   **軽巡・駆逐**: 大発動艇(八九式中戦車＆陸戦隊)、特二式内火艇、WG42などの対地装備。1隻は対空カットイン（秋月型等）推奨。
*   **特徴**: 高速統一を維持した基本の攻略編成です。難所であるCマスのル級flagship2隻の攻撃を耐えるため、キラ付けや煙幕の活用、または基地航空隊をCマスに送るなどの道中対策が効果的です。

#### ◆ 左ルート長陸奥タッチ編成（火力重視・推奨）
*   **編成**: 長門改二（旗艦） ＋ 陸奥改二 ＋ 秋津洲1 ＋ 駆逐2 ＋ 航巡（または軽巡）1
*   **ルート**: A → D（空襲） → C（通常/ル級） → F（空襲） → N（ボス）（道中4戦・低速混在可能）
*   **制空値**: 80〜100程度（秋津洲に水戦×3を装備させて制空を補助）
*   **特徴**: 長門改二・陸奥改二の一斉射（タッチ攻撃）をCマスまたはボスマスで発動し、敵を撃滅する非常に強力な編成です。秋津洲を組み込むことで、戦艦2隻編成かつ低速混在でありながら左最短ルートを通ることができます。

#### ◆ 右最短ルート編成（空母機動部隊任務向け）
*   **編成**: 戦艦1 ＋ 正規空母1 ＋ 軽巡1 ＋ 駆逐2 ＋ 揚陸艦1
*   **ルート**: M（対潜） → K（通常） → J（通常/空母棲姫） → N（ボス）（道中4戦・空襲回避）
*   **制空値**: 170〜200（Jマスのヲ級改や空母棲姫に対して優勢以上を確保）
*   **特徴**: 空母出撃制限のある任務（「機動部隊決戦」など）で右ルートを使用する際の最適編成です。揚陸艦（神州丸、あきつ丸等）を水戦・艦戦箱にすることで制空を稼ぎます。Mマスの対潜（ソ級）対策として、基地航空隊に東海を派遣するか本隊に先制対潜艦を組み込むのが推奨されます。

### 6. 特記事項・ギミック
*   **基地航空隊の運用**:
    *   通常は1部隊を**ボスマス（N）**に集中し、陸攻4（または陸戦1陸攻3）で敵砲台小鬼等の随伴を削ります。
    *   道中のCマス撤退が多い場合は、Cマスに航空隊（陸攻4）を送ることで道中を非常に安定させることができます（本隊の対地装備が充実している場合に有効）。
*   **対地特効のシナジー装備**:
    *   陸上型（離島棲姫、集積地棲姫、砲台小鬼）には、三式弾、大発動艇(八九式中戦車＆陸戦隊)、特二式内火艇、WG42、対地噴進砲が有効です。
    *   特に、駆逐・軽巡等に「大発動艇(八九式中戦車＆陸戦隊)」＋「特二式内火艇」をシナジー装備させることで、集積地棲姫に対して一撃で壊滅させるほどの超大ダメージを与えることが可能です。
