## 2-3. 東部オリョール海

### 1. 海域概要・特徴
南西諸島海域の3番目の海域です。
*   **作戦名**: オリョール哨戒
*   **クリア条件**: ボス（Nマス）でB勝利以上
*   **主な出現アイテム**: 燃料（Dマス、Iマス）、弾薬（Gマス、Hマス）
*   **特徴**: マップのあちこちに燃料や弾薬を獲得できる資源マスが配置されています。ランダム要素が多い海域ですが、条件を満たすことでボスマス（N）へ到達しやすくなる、または固定する編成を構成できます。多くの単発任務の指定海域でもあります。

---

### 2. ルート分岐法則
艦隊構成（水母や駆逐の隻数）によって、資源を拾いながらボス（N）へ向かう確率を大幅に引き上げることができます。

| 分岐点 | 進行マス | 条件 |
| --- | --- | --- |
| **スタート** | **A**（通常） / **B**（戦闘回避） / **C**（戦闘回避） | ・(潜水艦+潜水母艦)のみの艦隊で **C** 確定<br>・それ以外は **A** または **B** のランダム（各50%） |
| **C** | **D**（燃料） / **F**（通常） | ・Dマス寄りランダム（60%でD） |
| **D** | **F**（通常） / **G**（弾薬） | ・(水母+補給)1隻以上 ＋ (駆逐+海防)2隻以上で **G** 確定<br>・潜水艦2隻以上 ＋ 潜水母艦を含むと **G** 確定<br>・それ以外はランダム（(駆逐+海防)が多いほどG寄り） |
| **F** | **G**（弾薬） / **H**（弾薬） / **J**（通常） | ・(空母系+水母)1隻以上でJ寄り（約90%）ランダム<br>・それ以外はランダム（G: 25%、H: 35%、J: 40%） |
| **G** | **I**（燃料） / **K**（通常） | ・(軽巡+駆逐+海防)1隻以下で **K** 確定<br>・(水母+補給)1隻以上 ＋ (駆逐+海防)2隻以上でI寄り（65%）ランダム<br>・それ以外はK寄りランダム（駆逐海防が0隻ならK確定） |
| **J** | **L**（戦闘回避/逸れ） / **M**（輸送艦） / **N**（ボス） | ・軽巡1隻以上 ＋ 駆逐4隻以上で **N** 確定<br>・軽巡1隻 ＋ 重巡5隻（※航巡不可）で **N** 確定<br>・(戦艦級+空母系)6隻で **L** 確定、5隻でL寄り（85%）ランダム<br>・(戦艦級+空母系)2隻以下でN寄り（90%）ランダム |

### 3. おすすめ編成例
#### ◆ ボス到達率90%以上・バランス周回編成（任務・周回推奨）
*   **編成**: 正規空母（または軽空母） 2隻 ＋ 水上機母艦 1隻 ＋ 軽巡 1隻 ＋ 駆逐 2隻
*   **ルート**: スタートからランダム。
    *   【下（A）側出発時】: A → D（燃料） → G（弾薬） → (I) → K → N (ボス)
    *   【上（B）側出発時】: B → E → F → J → N (ボス)
*   **制空値**: 162以上（ボスマスで制空権確保）
*   **陣形**: 通常戦はすべて単縦陣
*   **特徴**: 水母1＋駆逐2の構成により、下ルートへ入った場合は燃料・弾薬を回収しつつ確実にボスへ到達します。上ルートへ入った場合も、空母2＋軽巡1を含むためJマスから90%の確率でボスマス（N）へ進めます。

#### ◆ ボスマス確定・軽量編成
*   **編成**: 軽巡 1隻 ＋ 駆逐 4隻 ＋ 自由枠 1隻（重巡/空母など）
*   **ルート**: どのルートを通っても、Jマスからボスマス（N）へ100%固定されます。
*   **特徴**: 確実にボスへ行きたい場合や、軽巡や駆逐艦の指定がある任務の攻略に向いています。

#### ◆ オリョクル編成（低コスト・潜水艦周回）
*   **編成**: 潜水艦/潜水空母 6隻（または潜水母艦 1隻 ＋ 潜水艦 5隻）
*   **ルート**: Cマス出発確定。D、G、I、Hなどの資源マスを複数踏みつつ、ボスマス（KまたはN）に到達します。
*   **特徴**: 弾薬や燃料の回収を兼ねた超省エネ周回編成です。第一期ほどの効率はありませんが、バケツ消費を完全に抑えながらデイリー任務をこなすのに有効です。

### 6. 特記事項・ギミック
*   **オリョクル（オリョールクルージング）**:
    *   第一期では「潜水艦のみでボスマス固定可能かつ無消費で資源が荒稼ぎできる」最重要周回スポットでしたが、第二期ではランダム逸れが発生するため、現在は主に任務消化（い号・ろ号）や低コストのレベリングを兼ねた周回手法として用いられます。
