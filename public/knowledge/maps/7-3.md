## 7-3. ペナン島沖

### 1. 海域概要・特徴
重巡洋艦や駆逐艦を中核とした軽量艦隊で挑む海域です。毎月1日0時にゲージが両方復活し、進行ルートも初期化されます（クリア勲章などの報酬はありません）。
*   **作戦名**: マラッカ海峡を抜けて
*   **難度**: ☆☆☆☆☆☆☆☆
*   **クリア条件**:
    *   **第一ゲージ（Eマス・ボス1）**: 旗艦の3回撃破で破壊。破壊後、マップ右半分と第二ボスマスが出現します。
    *   **第二ゲージ（Pマス・ボス2）**: 旗艦の4回撃破で海域クリア。
*   **特徴**:
    *   史実のペナン沖海戦をモチーフとしており、**特定の史実艦（羽黒、足柄、神風）**が強力なルート固定に関わります。
    *   戦艦や空母系は出撃不可、またはルートが大幅に逸れるため、基本的には巡洋艦以下による4〜6隻の軽量編成で攻略します。

---

### 2. ルート分岐法則
史実艦（羽黒・足柄・神風）の有無、隻数、速力によって進行ルートが決定されます。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **スタート** | ・空母系を含む、または(戦艦級を含み羽黒を含まない) → **B**（PT小鬼）<br>・【史実ペア】羽黒 ＋ 神風を含む → **C**（通常）<br>・【4隻以下】(重巡+軽巡+駆逐+海防)のみの艦隊 → **C**（通常・最短）<br>・それ以外は **B**（PT小鬼） |
| **C** | ・(戦艦級+空母系)1隻以上、または(駆逐+海防)0隻、または6隻編成 → **D**（通常）<br>・【最速統一】 → **I**（通常）<br>・【高速+統一】かつ (軽巡+駆逐)4隻以上 → **I**（通常）<br>・【史実ペア】羽黒 ＋ 神風を含み、かつ[軽巡または足柄を含む] → **I**（通常）<br>・【4隻以下】羽黒を含み、かつ(駆逐+海防)3隻 → **E（第一ボス）**<br>・【4隻以下】神風を含み、かつ(駆逐+海防)4隻 → **E（第一ボス）**<br>・それ以外は **D**（通常） |
| **G（第二出現後）** | ・【史実トリオ】羽黒 ＋ 足柄 ＋ 神風を含み、かつ[4隻以下、または高速+統一] → **P（第二ボス）**<br>・【史実トリオ】羽黒 ＋ 足柄 ＋ 神風を含み、かつ足柄を含む場合 → **P（第二ボス）**（※6隻高速統一でボス2確定）<br>・それ以外は **I**（通常）または **J**（対潜）へ逸れる |

*   **第一ゲージ最短ルート【ACE】**: 羽黒 ＋ 駆逐3（4隻編成・道中1戦）
*   **第二ゲージ最短ルート【ACDGP】**: 羽黒 ＋ 足柄 ＋ 神風 ＋ 駆逐2 ＋ 自由枠1（6隻編成・高速統一必須・道中2戦）

### 3. おすすめ編成例
#### ◆ 第一ゲージ：羽黒・駆逐最短編成（推奨・神風掘り）
*   **編成**: 羽黒1 ＋ 駆逐3
*   **ルート**: A → C（通常） → E（第一ボス）（道中1戦）
*   **陣形**: C:単縦陣 / ボス（E）:単縦陣
*   **特徴**: 最も簡単かつ軽量な第一ゲージ攻略編成です。道中通常戦1戦のみでボス（駆逐5隻）へ到達可能です。

#### ◆ 第二ゲージ：史実艦最短ルート編成（推奨・定番）
*   **編成**: 羽黒1 ＋ 足柄1 ＋ 神風1 ＋ 駆逐2 ＋ 自由枠1（水母または雷巡）
*   **ルート**: A → C（通常） → D（通常） → G → P（第二ボス）（道中2戦・高速統一）
*   **陣形**: C:単縦陣 / D:単縦陣 / ボス（P）:単縦陣
*   **特徴**: 羽黒・足柄・神風の3隻を起用することで、6隻編成でありながら道中通常戦2戦のみで第二ボスへ向かえます。自由枠には甲標的による先制雷撃と水爆での制空補助ができる三米改二特（水母）や日進甲が最も推奨されます。

### 6. 特記事項・ギミック
*   **神風のドロップ狙い**:
    *   第二ゲージ固定には神風が必要なため、未所持の場合は第一ゲージ破壊周回（羽黒＋駆逐3）で神風をドロップさせてから第二ゲージを攻略するのが効率的です。
