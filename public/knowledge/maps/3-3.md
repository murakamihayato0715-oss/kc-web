## 3-3. アルフォンシーノ方面

### 1. 海域概要・特徴
北方海域の第3海域。空母・戦艦が複数出現し、これまでより難度が上がります。
*   **作戦名**: アルフォンシーノ方面進出
*   **難度**: ☆☆☆☆☆☆☆
*   **クリア条件**: ボスB勝利以上
*   **主な入手アイテム**: 家具箱（中）・家具箱（大）
*   **出撃解放条件**: 2-4クリア **かつ** 3-2クリア
*   **特徴**: ボス到達のために**渦潮（Cマス）を必ず経由**しなければならない特殊なマップです。電探の装備が必須です。攻略ルートは正規空母ありの**北ルート**（安定・推奨）と、正規空母なしの**南ルート**（羅針盤が複雑）の2択です。

---

### 2. ルート分岐法則
正規空母の有無で北ルート（Cマス経由）か南ルート（Bマス経由）かが決まります。

| 分岐点 | 条件（上から優先） |

| **A** | 正規空母1以上 → C（北ルートへ）<br>(戦艦+空母)4以上 → C<br>(戦艦+空母)1かつ軽巡1かつ駆逐4 → C<br>その他 → B（南ルートへ） |
| **B** | 潜水1以上 → ランダムでDかF<br>(戦艦+空母)1以下 → F<br>(戦艦+空母)2かつ(駆逐+海防)2以上 → F<br>その他 → 50%でD / 50%でF |
| **C（渦潮）** | (駆逐+海防)1以下 → E<br>正規空母2以上 → E<br>(戦艦+空母)3以上 → E<br>(戦艦+空母)2 → G<br>(戦艦+空母)1かつ軽巡1かつ駆逐4 → G<br>その他 → E |
| **F** | 駆逐1以下 → G<br>戦艦3以上 → G<br>潜水1以上 → ランダムでHかJ<br>軽巡1以上 → J<br>航巡1以上 → J<br>水母1以上 → J<br>その他 → 50%でJ / 50%でH |

### 3. おすすめ編成例
#### ◆ 初攻略・北3戦ルート（推奨）
*   **編成**: 正規空母1 ＋ 軽空母1 ＋ 重巡2 ＋ 駆逐2
*   **ルート**: A → C（渦潮） → G → M（ボス）（道中3戦）
*   **制空値**: 目安160以上（確保156 優勢78）
*   **陣形**: 全戦単縦陣
*   **特徴**: 渦潮あり。電探を3〜5隻に装備して燃料喪失を軽減。(戦艦+空母)が4以上になるとGマスでIマスへランダム逸れが発生するため注意。

#### ◆ 重量編成・北4戦ルート（道中安定向け）
*   **編成**: 正規空母1 ＋ (戦艦+空母)計3 ＋ 自由3
*   **ルート**: A → C（渦潮） → E → G → M（ボス）（道中4戦）
*   **特徴**: 制空値と艦隊火力を両立しやすく道中安定度が高い。複数の雷巡を入れた先制攻撃強化も可能。消費が重めになる点に注意。

#### ◆ 家具コイン回収（D/Hマス狙い）
*   **編成例**: 軽空母2〜3 ＋ 駆逐2 ＋ 潜水1〜2
*   **ルート**: Dマス（家具箱中）またはHマス（家具箱大）経由
*   **特徴**: 1周あたり期待値約400〜440家具コイン。ボスS勝利と並行しながら家具コインを稼ぐ際に活用。

### 6. 特記事項・ギミック
*   ギミックなし。クリア条件はボスB勝利以上。
*   **渦潮（Cマス）必須**: 通常海域で渦潮が強制の稀なマップ。電探を3〜5隻に装備して燃料喪失を軽減すること。
*   **Fマス（強うずしお）**: 南ルートで出現。電探が少ない状態で強うずしお発生時はほぼ燃料ゼロになる危険あり。南ルートは上級者向け。
*   ウィークリー「**敵北方艦隊主力を撃滅せよ！**」では**改修資材(ネジ)が3本**入手可能。継続的なクリアを推奨。
