## 7-5. ジャワ島沖

### 1. 海域概要・特徴
通常海域で唯一「トリプルゲージ」と「ルート開放ギミック」を備える Extra Operation（EO）海域です。
*   **作戦名**: スラバヤ沖海戦・バタビア沖海戦
*   **難度**: ☆☆☆☆☆☆☆☆☆☆☆
*   **クリア条件**:
    *   **第一ゲージ（Kマス・ボス1）**: 旗艦の2回撃破で破壊。破壊後、第二段階が出現します。
    *   **第二ゲージ（Qマス・ボス2）**: MマスでS勝利1回（ギミック）を達成し、ボス（Qマス・陸上型）の旗艦を3回撃破で破壊。破壊後、第三段階が出現します。
    *   **第三ゲージ（Tマス・ボス3）**: 旗艦の3回撃破で海域クリア。
*   **特徴**:
    *   基地航空隊および支援艦隊は出撃できません。
    *   スラバヤ沖海戦・バタビア沖海戦に参加した史実艦に、強力なダメージ特効補正（最大1.15倍）が存在します。
    *   毎月1日0時にゲージが復活し、初期化されます（勲章×1、特別戦果＋170のクリア報酬あり）。

---

### 2. ルート分岐法則
艦隊構成（戦艦、空母の隻数）、速力、および索敵スコア（分岐点係数4）によって決定されます。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **スタート** | ・高速+以上統一、または(正規空母1以下かつ軽空2以下かつ戦艦2以下かつ重巡2以下かつ軽巡1以上かつ(駆逐+海防)3以上) → **D**（通常）<br>・それ以外は **C**（通常）へ逸れる |
| **D** | ・最速統一、または高速+統一 → **F**（能動分岐）<br>・正規空母2隻以上、空母系3隻以上、または(戦艦級+正規空母+重巡級)3隻以上 → **E**（PT小鬼）<br>・戦艦級2隻の場合かつ[(高速統一かつ(駆逐+海防)2以上)または(駆逐+海防)3以上] → **F**（能動分岐）<br>・(軽巡+駆逐+海防)2隻以上 → **F**（能動分岐）<br>・それ以外は **E**（PT小鬼） |
| **I** | ・【索敵スコアチェック（分岐点係数4）】<br>　・索敵スコア **59以上** で **M**（通常）進行、52以下でL（通常） |
| **P（第三出現後）** | ・索敵スコア **63以上** でかつ「正規空母0、(戦艦+軽空母)1以下、(戦艦+重巡級)2以下、軽巡1以上」 → **T（第三ボス）**<br>・最速統一 → **T（第三ボス）**<br>・それ以外は **R**（通常・逸れ） |

*   **最短ルート【A-B-D-F-J-O-Q(またはT)】**: A（空襲） → B（対潜） → D（水上） → F（能動） → J（水上） → O（能動） → 各ボス

### 3. おすすめ編成例
#### ◆ 第一・第二・第三ゲージ共通：最短ルート軽量編成（推奨）
*   **編成**: 航空戦艦1（日向改二等） ＋ 航巡1（最上改二特等） ＋ 軽巡1 ＋ 駆逐3
*   **ルート**: A → B（対潜） → D → F → J → O → 各ボス（低速可・索敵スコア63以上）
*   **装備**:
    *   **航空戦艦**: 主砲×2 ＋ ヘリ（S-51J等） ＋ 水偵（先制対潜と2巡化）
    *   **航巡**: 甲標的 ＋ 主砲×2 ＋ 水戦1（先制雷撃と制空権確保）
    *   **軽巡・駆逐**: 
        *   *第一・第三ゲージ*: 魚雷カットイン装備（特効のある敷波、暁など）
        *   *第二ゲージ*: 対地装備（大発動艇、内火艇、WG42など）および対PT小鬼装備（見張員、小口径主砲＋電探など）
*   **特徴**: 全てのゲージおよびギミック（MマスS勝利）を同一の艦種構成で攻略できる定番の編成です。水爆や航空機はボスのト級に全滅させられやすいため、制空用には水戦を推奨します。

#### ◆ 第三ゲージ：高速戦艦2隻・最速統一最短編成（榛名任務用）
*   **編成**: 高速戦艦2（榛名改二乙/丙 ＋ 金剛改二丙など） ＋ 軽空母1 ＋ 軽巡1 ＋ 駆逐2
*   **ルート**: A → B → D → F → J → O → P → T（第三ボス）（※要最速統一）
*   **特徴**: 任務用などで戦艦2隻を編成しつつ、Rマスの強力なヌ級改・ネ級部隊を回避する最速統一編成です。缶・タービンの装備枠が多くなるため、4スロ艦や高速A群の駆逐艦（島風、タシュケントなど）を推奨します。
