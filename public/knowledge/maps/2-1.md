## 2-1. 南西諸島近海

### 1. 海域概要・特徴
南西諸島海域の最初の海域です。敵空母や戦艦が本格的に出現し始めます。
*   **作戦名**: 南西諸島哨戒
*   **クリア条件**: ボス（Hマス）でB勝利以上
*   **主な出現アイテム**: 鋼材（Bマス）、高速建造材（Eマス）
*   **特徴**: 敵空母・戦艦が出現し始めるため、十分な制空値の確保が重要になります。編成の艦種・隻数および速力によって、最短1戦でボスマスへ到達するルート（C → E → H）を固定可能です。

---

### 2. ルート分岐法則
空母系の数や特定の軽量編成、高速統一によってボス（H）へ固定ルートを構成できます。

| 分岐点 | 進行マス | 条件 |
| --- | --- | --- |
| **スタート / C** | **B**（鋼材） / **D** / **E**（高速建造材） | ・空母系3隻以上、または航空戦艦2隻以上、または「補給艦1隻以上かつ潜水艦0隻」で **B** 確定<br>・航空戦艦1隻編成の場合：(水母/潜母)1隻以上でB寄りランダム、それ以外はBまたはDランダム<br>・航空戦艦0隻かつ(水母/潜母)1隻以上で **E** 確定<br>・それ以外はDまたはEのランダム（各50%） |
| **D** | **H**（ボス） | 固定 |
| **E**（高速建造材） | **D** / **F** / **H**（ボス） | ・(戦艦級+空母系)5隻以上で **F** 確定<br>・【6隻編成時】(駆逐+海防)6隻、または軽巡1隻＋(駆逐+海防)5隻、または【高速統一】かつ軽巡1隻＋駆逐4隻で **H** 確定（それ以外はD）<br>・【5隻以下編成時】(駆逐+海防)5隻、または軽巡1隻＋(駆逐+海防)4隻、または【高速統一】かつ軽巡1隻＋駆逐3隻で **H** 確定（それ以外はD寄りランダム） |
| **F** | **G**（戦闘回避） / **H**（ボス） | ・(戦艦級+空母系)5隻以上で **G** 確定<br>・駆逐3隻以上、または軽巡1隻以上＋駆逐2隻以上で **H** 確定<br>・それ以外はH寄りランダム（60%でH） |

### 3. おすすめ編成例
#### ◆ 最短道中1戦ルート編成（周回・任務向け）
*   **編成**: 軽巡 1隻 ＋ 駆逐 4隻 ＋ 水上機母艦 1隻（または潜水母艦 1隻）
*   **ルート**: C → E → H (道中1戦のみ)
*   **速力**: **高速統一**（低速水母を採用する場合はタービン＋強化型コブラによる高速化が必要）
*   **制空値**: 81以上（ボスマス航空優勢）または162以上（制空権確保）
*   **陣形**: すべて単縦陣
*   **特徴**: 高速統一かつ軽量編成にすることで、戦闘がCマスの1回のみでボスに到達します。水母の開幕爆撃と甲標的による先制雷撃で敵を減らして進む、バケツ消費を抑えやすい周回用編成です。

#### ◆ 3戦安定・初攻略編成（戦艦・空母運用）
*   **編成**: 正規空母 1隻 ＋ 軽空母 1隻 ＋ 重巡 2隻 ＋ 軽巡 2隻（※戦艦2隻まで交代可、航戦は不可）
*   **ルート**: C → D → H または C → E → D → H (道中2戦)
*   **制空値**: 81〜162以上（ボスマス優勢〜確保）
*   **陣形**: すべて単縦陣
*   **特徴**: 空母2隻の開幕航空攻撃で道中の敵を削り、ボスマスでは弾着観測射撃による火力を叩き込む、安定感の高い攻略編成です。

#### ◆ 潜水艦6隻周回編成（コスト節約・デイリー消化用）
*   **編成**: 潜水艦/潜水空母 6隻
*   **ルート**: C → E → D → H（道中2戦）
*   **特徴**: 燃費が非常に良く、ボスマスでの南西デイリー任務やい号作戦（敵空母撃破）の同時消化に向いています。後期型駆逐艦などの対潜攻撃による被弾大破に注意してください。

### 6. 特記事項・ギミック
*   **司令部レベルによる敵難易度変化**:
    *   司令部レベルが上がると（特にLv80〜120）、ボスマスや道中の敵にelite級や駆逐後期型が配備され、制空値も上昇します。
*   **高速建造材（バーナー）の獲得**:
    *   Eマスを踏むことで高速建造材（バーナー）を確実に獲得できます。昨今では特定艦（吹雪改三など）の改装用やイベントの友軍艦隊要請で大量消費するため、水母1を含む編成で「C→E→撤退」や「C→E→D→撤退」を繰り返すバーナー回収周回が行われることがあります。
