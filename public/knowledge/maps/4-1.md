## 4-1. ジャム島沖

### 1. 海域概要・特徴
西方海域の入口。4-1から潜水艦の敵が新たに出現するようになります。
*   **作戦名**: ジャム島攻略作戦
*   **難度**: ☆☆☆☆☆
*   **クリア条件**: ボスB勝利以上
*   **主な入手アイテム**: 燃料（Bマス）
*   **出撃解放条件**: 鎮守府海域・南西諸島海域・北方海域3-1クリア
*   **特徴**: 初手がA/Cのランダムスタート。(戦艦+空母)の数とルートの関係が複雑です。**Eマス（輸送船団）に逸れるとボス不達**になるため、(戦艦+空母)は2隻以下が基本となります。ボスマスは空母なしまたは軽母1隻程度の比較的易しい編成です。

---

### 2. ルート分岐法則
初手ランダムが最大の特徴。(戦艦+空母)の数と駆逐・軽巡・重巡の配分でルートが決まります。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **始点** | AかCにランダム（50:50） |
| **C** | (戦艦+空母)5以上 → E（逸れ・ボス不達）<br>(戦艦+空母)4 → 70%でE / 30%でF<br>(戦艦+空母)3 → 50%でE / 50%でF<br>(戦艦+空母)2以下 → F |
| **D** | (戦艦+空母)5以上 → H<br>潜水1以上 → 30%でG / 70%でH<br>(戦艦+空母)4 → G<br>(戦艦+空母)1〜3：(駆逐+海防)4以上 → H、3以上 → G、2時は軽巡0→G・重巡1以上かつ(重巡+軽巡+練巡)3→H・その他ランダム、1以下 → G<br>(戦艦+空母)0：(駆逐+海防)2以上 → H、1以下 → G |
| **F** | (戦艦+空母)1以上 → D<br>(駆逐+海防)6 → H<br>軽巡1以上かつ(駆逐+海防)4以上 → H<br>その他 → D |
| **H** | 潜水1以上 → I（行き止まり）<br>(戦艦+空母)5以上 → I<br>(駆逐+海防)2以上かつ(戦艦+空母)1以下 → J（ボス）<br>その他 → ランダムでIかJ |

**主なルート固定条件**

| 編成 | ルート | 特徴 |
| --- | --- | --- |
| 正規空母1・重巡2・軽巡1・駆逐2 | CFDHJ / ABDHJ（道中3〜4戦） | G（戦艦タ級）回避。初攻略推奨 |
| 正規空母2・重巡3・駆逐1 | CFDGJ / ABDGJ（道中4戦） | 開幕火力で戦艦タ級を制圧 |
| 駆逐4・(軽巡か水母か雷巡)2 | CFHJ / ABDHJ（道中2〜3戦） | (戦艦+空母+重巡)不可。軽量最短 |

### 3. おすすめ編成例
#### ◆ 初攻略・G回避編成（正規空母1重巡2軽巡1駆逐2）
*   **編成**: 正規空母1 ＋ 重巡2 ＋ 軽巡1 ＋ 駆逐2
*   **ルート**: C/A → F → D（対潜） → H → J（ボス）（道中3〜4戦）
*   **制空値**: 目安75以上（確保72 優勢36）
*   **陣形**: 道中は単縦陣、Dマス（潜水艦）は単横陣
*   **特徴**: 戦艦タ級2隻が出るGマスを回避してボスへ。Dマスの潜水艦対策にソナーを軽巡・駆逐に装備すること。条件：(戦艦+空母)1以下・重巡1以上・軽巡1・(重巡+軽巡+練巡)3・(駆逐+海防)2以上。

#### ◆ 重量編成・G突破（正規空母2重巡3駆逐1）
*   **編成**: 正規空母2 ＋ 重巡3 ＋ 駆逐1
*   **ルート**: C/A → F → D（対潜） → G（戦艦タ級） → J（ボス）（道中4戦）
*   **制空値**: 目安96以上（確保72 優勢36）
*   **特徴**: 空母の開幕航空戦でGマスの戦艦タ級を先制する。軽巡の練度が不足している場合や、開幕火力で押し切りたい場合に有効。雷巡を加えて先制雷撃を活用しても良い。

### 6. 特記事項・ギミック
*   ギミックなし。クリア条件はボスB勝利以上。
*   Dマス（対潜戦）は通常消費。ソナー・爆雷の装備と単横陣を使用することで効率的に対処可能。
*   **レベリング用途**: 重量編成（(戦艦+空母)2・駆逐1以下）でのG経由3戦周回はタ級elite以外の敵が弱く、バケツ節約レベリングに適しています。初戦の大破撤退率が比較的低いのも利点です。
*   **注意**: (戦艦+空母)5以上だとDHIが確定してボスに到達不可。4以下に抑えること。
