## 1-3. 製油所地帯沿岸

### 1. 海域概要・特徴
1-2クリア後に解放される海域です。
*   **作戦名**: 海上護衛作戦
*   **クリア条件**: ボス（Jマス）でB勝利以上
*   **主な出現アイテム**: 燃料（Dマス、Gマス）
*   **特徴**: 敵に「重巡洋艦」や「戦艦」が初登場する海域です。また、マップ上に資材（燃料）を喪失する「うずしお」マスが初登場します。

---

### 2. ルート分岐法則
特定の艦種構成や速力によって、うずしおを回避してボスマス（J）へ固定できます。

| 分岐点 | 進行マス | 条件 |
| --- | --- | --- |
| **スタート** | **A** / **C** | ・補給艦または水上機母艦を含むと **A** 確定<br>・空母系を含むと **C** 確定<br>・それ以外はランダム（各50%） |
| **A** | **D**（燃料） / **E** | ・補給艦1隻以上で **D** 確定<br>・海防艦4隻以上で **D** 確定<br>・水母含むとDマス寄り（80%）のランダム<br>・(駆逐+海防)4隻以上でDマス寄り（80%）のランダム<br>・潜水艦を含むと **E** 確定<br>・それ以外はランダム（各50%） |
| **F** | **H**（うずしお） / **J**（ボス） | ・正規空母または低速戦艦を含むと **H** 確定<br>・駆逐4隻以上で **J** 確定<br>・航巡1隻以上 ＋ 駆逐2隻以上で **J** 確定<br>・(軽巡+練巡)1隻以上 ＋ (駆逐+海防)4隻以上で **J** 確定<br>・【高速統一】以上でJ寄り（75%）のランダム<br>・それ以外はH寄りランダム |
| **H**（うずしお） | **G**（燃料） / **I** / **J**（ボス） | ・補給艦1隻以上で **G** 確定<br>・水母または航巡を含むと **J** 確定<br>・(軽巡+練巡)1隻以上 ＋ 駆逐2隻以上で **J** 確定<br>・それ以外はランダム（IまたはJ、I寄り） |

### 3. おすすめ編成例
#### ◆ ボス最短攻略/周回編成（うずしお回避・軽量編成）
*   **編成**: 軽空母 2隻 ＋ 駆逐 4隻
*   **ルート**: C → F → J (ボス最短) または A → D → B → E → F → J
*   **制空値**: 52以上で制空権確保（強編成の優勢/確保に必要な制空値は不要ですが、開幕航空戦の手数を最大化するため戦闘機・爆撃機を適切に搭載してください）
*   **陣形**: すべて単縦陣
*   **特徴**: 空母の開幕航空攻撃で道中・ボス戦ともに敵艦を撃破し、安定した周回が可能です。駆逐艦4隻を編成することでうずしお（H）を回避しボスマスへ固定できます。
*   **アドバイス**: 軽空母の射程を副砲や中射程艦載機で伸ばすことで、駆逐艦より先に砲撃戦を行うよう調整すると手数が安定します。

#### ◆ 燃料回収周回（三重クル / ADBE撤退）
*   **編成**: 補給艦 1隻 ＋ 駆逐 5隻（または駆逐4隻＋潜水艦1隻）
*   **ルート**: A → D（燃料） → B（気のせい） → E（通常戦） → 撤退
*   **装備**: 各艦にドラム缶や大発動艇、特二式内火艇をフル装備（大発系1スロットにつき獲得燃料+3、ドラム缶は+2）
*   **特徴**: 補給艦を1隻編成することでスタートからA・Dへ確定進行します。Dマスで燃料を回収した後、Eマスで戦闘を行い、戦闘終了後に撤退することで効率よく燃料を稼ぐことができます（三重クル）。
*   **注意**: E戦後の誤進撃に十分注意してください。

### 6. 特記事項・ギミック
*   **敵戦艦の出現と「砲撃戦2巡化」**:
    *   戦艦ル級がボスマスに出現します。戦艦が敵味方いずれかに存在すると、昼戦の砲撃戦が自動的に2巡します。被弾が増えやすいため、ボス戦では夜戦も視野に入れ、B勝利以上（旗艦撃破または敵艦3隻以上撃破）を狙いましょう。
*   **うずしおマスの対策**:
    *   Hマスのうずしおでは燃料を喪失します。電探を装備した艦を多く編成することで、資材喪失量を軽減できます。五十鈴改（Lv12で改造可）などが持参する「21号対空電探」などが手軽でおすすめです。
