## 4-4. カスガダマ島

### 1. 海域概要・特徴
西方海域の第4海域（ラストノーマル海域）。戦力ゲージがあり、育成と周回を兼ねられる定番マップです。
*   **作戦名**: カスガダマ沖海戦
*   **難度**: ☆☆☆☆☆☆☆☆
*   **クリア条件**: ボス旗艦（装甲空母姫またはヌ級FL）を4回撃沈（戦力ゲージ100%→0%）
*   **主な入手アイテム**: 燃料（Cマス）、鋼材（Jマス）
*   **特徴**:
    *   **中央最短ルート（AEIK）**: 道中に敵戦艦・空母が出ない3戦ルート。正規空母2＋駆逐2でボスへの最短を確定できます。
    *   **Eマス**: 対潜戦（燃料8%消費・弾薬消費なし）。潜水ヨ級が4隻出現します。
    *   **ボス（Kマス）**: 装甲空母姫（パターン2・3）またはヌ級FL（パターン1）がランダム。随伴に潜水ヨ級が入るパターンもあるため対潜装備を1本は持参推奨。
    *   **育成海域として優秀**: Eマスで駆逐にMVPを取らせやすく、ボスへの到達率も高い。レア艦（翔鶴・瑞鶴・大鯨・長門）もドロップします。

### 2. ルート分岐法則
中央最短（AEIK）固定が攻略の基本です。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **始点** | (駆逐+海防)2以上 → A<br>その他 → 50%でA / 50%でB |
| **B** | (戦艦+空母)4以上 → A<br>その他 → 高確率でD / 低確率でF |
| **E** | (戦艦+正空)4以上 → G<br>海防3以上 → C（燃料マス）<br>海防2かつ(補給+潜母)1以上 → C<br>(駆逐+海防)2以上かつ(軽巡+重巡)1以上 → **I（ボス前へ）**<br>(駆逐+海防)2以上かつ(戦艦+空母)4以上 → 80%でG<br>(駆逐+海防)2以上かつ(戦艦+空母)3以下 → 65%でI<br>その他 → ランダム |
| **I** | (駆逐+海防)2以上かつ正空2 → **K（ボス）**<br>(駆逐+海防)2以上かつ重巡級2 → K<br>(駆逐+海防)2以上かつ正空0かつ軽巡1以上 → K<br>(駆逐+海防)2以上のその他 → 75%でK<br>その他 → 50%でK |

**主なルート固定条件（すべてAEIK）**

| 編成 | ルート固定条件 |
| --- | --- |
| 戦艦1・正規空母2・重巡1・(駆逐+海防)2 | 正空2＋(重巡+軽巡)1でE→I固定、I→K固定 |
| 正規空母2・重巡2・(駆逐+海防)2 | 正空2でE→I固定（重巡2でI→K固定） |
| 正規空母2・軽空母1・(重巡か軽巡)1・(駆逐+海防)2 | 正空2でE→I固定、I→K固定 |
| 軽空母3・軽巡1・(駆逐+海防)2 | 軽巡1でE→I固定（正空0・軽巡1以上でI→K固定） |

### 3. おすすめ編成例
#### ◆ 定番編成（正規空母2重巡1戦艦1駆逐1海防1）
*   **編成**: 正規空母2 ＋ 重巡1 ＋ 戦艦1 ＋ 駆逐1 ＋ 海防1
*   **ルート**: A → E（対潜） → I → K（ボス）（道中3戦）
*   **制空値**: 目安80以上（確保144 優勢72）
*   **陣形**: Eマス単横陣、他は単縦陣
*   **特徴**: 戦艦で砲撃2巡化し道中安定。海防艦には対潜先制爆雷が可能な艦を推奨（無ければソナー+爆雷の駆逐でも可）。

#### ◆ 周回・省エネ編成（正規空母2軽空母1軽巡1駆逐2）
*   **編成**: 正規空母2 ＋ 軽空母1（護衛空母推奨） ＋ 軽巡1 ＋ 駆逐2
*   **ルート**: A → E（対潜） → I → K（ボス）（道中3戦）
*   **制空値**: 目安80以上
*   **特徴**: 戦艦を外しボーキ消費を抑えた周回向け編成。軽空母に先制対潜可能な護衛空母（大鷹型など）を推奨。射程調整に彩雲か彗星二二型(六三四空)を活用すると空母が先に砲撃できる。

### 6. 特記事項・ギミック
*   ゲージ破壊後は復活なし（恒久クリア）。
*   **育成向け海域**: 道中に戦艦・空母が出ないためAEIK周回が安定しやすい。EマスMVPを駆逐に取らせることで1周あたり旗艦1900〜2200exp（駆逐）を確保可能。
*   **装甲空母姫について**: 制空値48（確保144）で対処可。耐久は高いが装甲はタ級eliteより低め。潜水ヨ級随伴パターン（P3）では単縦陣でも対潜装備1本あれば対処可能。
*   **カスクル（AEC）**: 海防艦2＋(海防/補給/潜母)1＋自由3で燃料（Cマス）回収。平均燃料105の収益。
