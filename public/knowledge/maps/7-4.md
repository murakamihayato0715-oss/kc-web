## 7-4. 昭南本土航路

### 1. 海域概要・特徴
対潜戦および空襲戦がメインの海域で、通常海域でありながら**基地航空隊**を1部隊出撃させることができます。
*   **作戦名**: ヒ船団海上護衛作戦
*   **難度**: ☆☆☆☆☆☆☆☆☆
*   **クリア条件**: ボス（Pマス）旗艦「ヒ船団棲姫」を5回撃破（戦力ゲージクリア後は復活なし）
*   **特徴**:
    *   ボスの「ヒ船団棲姫」は極めて耐久・装甲が高く（最終形態は装甲201〜231）、通常海域屈指の難関ボスです。
    *   正規空母および低速戦艦を1隻でも編成するとボスマスに到達できなくなります。航空戦艦や軽空母、対潜能力の高い海防艦や駆逐艦を中心とした護衛艦隊で挑みます。
    *   毎月1日0時にゲージが復活します（勲章などのクリア報酬はありません）。

---

### 2. ルート分岐法則
艦隊内の艦種（戦艦・正規空母の禁止）、海防艦・駆逐艦の隻数、索敵スコア（分岐点係数4）によって決定されます。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **スタート** | ・戦艦、正規空母、または潜水艦を含む → **C**（対潜）<br>・重巡級2隻以上、軽巡級2隻以上 → **C**（対潜）<br>・(航空戦艦+軽空母+あきつ丸)3隻以上 → **C**（対潜）<br>・[あきつ丸 ＋ 海防2 ＋ (駆逐or海防)1]、または(駆逐+海防)3隻以上、または海防2隻以上 → **A**（通常・気のせい）<br>・それ以外は **C**（対潜） |
| **C** | ・戦艦、正規空母、潜水艦のいずれかを含む、または(軽空母+あきつ丸)3隻以上 → **D**（空襲）<br>・(駆逐+海防)4隻以上、または[練巡1 ＋ (駆逐+海防)3以上]、または海防3隻以上 → **E**（空襲）<br>・それ以外は **D**（空襲） |
| **E** | ・[海防4 ＋ (補給or揚陸)1 ＋ (大鷹型or補給or揚陸or駆逐or海防)1]の6隻編成 → **G**（対潜）<br>・それ以外は **J**（通常/対潜混合） |
| **J** | ・【索敵スコアチェック（分岐点係数4）】<br>　・索敵スコア **37以上** で **P（ボス）** 進行（※33未満でK逸れ、33〜37でランダム） |

*   **準最短ルート【A-B-E-J-L-P】**: A（気のせい） → B（対潜） → E（空襲） → J（通常/対潜） → L（通常/対潜） → P（ボス）
*   **準最短ルート【C-E-J-L-P】**: C（対潜） → E（空襲） → J（通常/対潜） → L（通常/対潜） → P（ボス）

### 3. おすすめ編成例
#### ◆ 準最短バランス編成（攻略・周回推奨）
*   **編成**: 航空戦艦1 ＋ 軽空母1 ＋ 軽巡1 ＋ 駆逐1 ＋ 海防艦2
*   **ルート**: A → B（対潜） → E（空襲） → J（通常/対潜） → L（通常/対潜） → P（ボス）（道中4戦・要索敵37以上）
*   **陣形**: B:単横陣 / E:輪形陣 / J:単横陣（または梯形陣） / L:単横陣 / ボス（P）:単横陣（※警戒陣を使用できる時期は警戒陣推奨）
*   **制空値**: 180〜220程度（Eマスやボスマスで航空優勢を狙う）
*   **装備**:
    *   **航空戦艦（伊勢改二等）**: 主砲×2 ＋ 艦戦×2 ＋ 偵察機/徹甲弾（道中2巡化および制空補助）
    *   **軽空母（大鷹型など護衛空母）**: 艦攻（対潜値付き） ＋ 艦戦 ＋ 彩雲（T字不利回避）
    *   **軽巡・駆逐・海防**: 全員に**対潜先制爆雷攻撃**（ソナー＋投射機＋爆雷の3点シナジー）を装備。駆逐1隻はフィニッシャー用に魚雷カットイン装備推奨。
*   **特徴**: 道中の被弾リスクを最小限に抑えつつ、ボスのヒ船団棲姫の撃破を狙う基本の攻略編成です。海防艦には強力な特効が存在します。

### 6. 特記事項・ギミック
*   **海防艦・練習巡洋艦の特効**:
    *   J, L, P（ボス）マスにおいて、海防艦（ボスで**1.33倍**）、練習巡洋艦（ボスで**1.23倍**）の与ダメージ特効補正が存在します。
*   **基地航空隊の活用**:
    *   基地航空隊（対潜哨戒機・**東海シリーズ**）をボスマス（P）またはJマスに派遣することで、厄介なソ級flagshipを先制撃破し、雷撃事故を減らすことができます。
*   **最終形態の難度上昇**:
    *   ゲージ破壊直前（最終形態）になると、道中のJマスが駆逐入りの単縦陣パターンで固定され、ボスの装甲も最大231に跳ね上がるため、難易度が爆発的に上昇します。任務達成を狙う場合は、ゲージ破壊後（削り編成に戻る）またはゲージリセット後に挑戦するのがおすすめです。
