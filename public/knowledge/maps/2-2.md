## 2-2. バシー海峡

### 1. 海域概要・特徴
南西諸島海域の2番目の海域です。
*   **作戦名**: 柳作戦
*   **クリア条件**: ボス（Kマス）でB勝利以上
*   **主な出現アイテム**: ボーキサイト（Aマス、Fマス）、高速建造材（Jマス）
*   **特徴**: 敵に戦艦ル級が出現し、司令部Lv上昇に伴いelite化するため道中・ボスマスともに大破リスクが高まります。一方で、水上機母艦（水母）や海防艦を適切に編成することで、最短道中1戦のみでボスマス（K）へ固定することができます。

---

### 2. ルート分岐法則
水母や潜水母艦の編成がスタートからの羅針盤固定に必須となります。

| 分岐点 | 進行マス | 条件 |
| --- | --- | --- |
| **スタート / C** | **B**（輸送艦/ボーキ） / **D** / **E**（通常） | ・空母系3隻以上、または航空戦艦2隻以上、または「補給艦1隻以上かつ潜水艦0隻」で **B** 確定（バシクルルート）<br>・航空戦艦1隻編成の場合：(水母/潜母)1隻以上でB寄り（70%）ランダム、それ以外はB/Dランダム<br>・航空戦艦0隻かつ(水母/潜母)1隻以上で **E** 確定（ボスルート）<br>・それ以外はDまたはEのランダム（各50%） |
| **E**（通常） | **F**（ボーキ） / **G**（通常戦） / **K**（ボス） | ・(戦艦級+空母系)4隻以上で **G** 確定<br>・海防艦2隻以上で **F** 確定<br>・(戦艦級+空母系)3隻でG寄り（70%）、2隻でG/Kランダム（50%）、1隻でK寄り（70%）ランダム<br>・(戦艦級+空母系)0隻かつ【高速統一】かつ軽巡1隻以上かつ(駆逐+海防)2隻以上で **K** 確定<br>・(駆逐+海防)3隻以上かつ潜水母艦を含むと **F** 確定 |
| **G**（通常戦） | **H**（戦闘回避） / **K**（ボス） | ・空母系を含む、または駆逐0隻で **H** 確定<br>・それ以外は基本K寄り（70%）ランダム |
| **H** | **I** / **J**（高速建造材） / **K**（ボス） | ・(空母系+航巡+水母)1隻以上で **K** 確定<br>・(駆逐+海防)2隻以上でJまたはKへ進行 |

### 3. おすすめ編成例
#### ◆ 道中1戦ボス確定編成（最安定・大型艦入り）
*   **編成**: 正規空母（または軽空母） 2隻 ＋ 軽巡 1隻 ＋ 水上機母艦 1隻 ＋ 海防艦 2隻
*   **ルート**: C → E → F（ボーキ） → H → K (道中1戦ボス確定・Fマスでボーキ回収可能)
*   **制空値**: 81以上（優勢）または162以上（確保）
*   **陣形**: Eマス（単単縦陣）／ Kマス（単縦陣）
*   **特徴**: 空母2隻の開幕航空火力で敵を制圧しつつ、海防艦2隻と水母1隻を入れることで道中1戦ボス確定となります。Fマスでボーキサイトも回収できる非常に優れた周回・攻略編成です。

#### ◆ 道中1戦ボス確定編成（高速統一水雷・巡洋艦主軸）
*   **編成**: 重巡（または航巡） 2隻 ＋ 軽巡 1隻 ＋ 駆逐 2隻 ＋ 水上機母艦 1隻
*   **ルート**: C → E → K (道中1戦ボス確定)
*   **速力**: **高速統一**
*   **特徴**: 海防艦を使わずに重巡/航巡の火力でボスを叩く編成です。低速の水母を採用する場合は高速化が必須となります。

#### ◆ 補給艦撃破＆ボーキサイト回収（バシクル周回 / 1戦撤退）
*   **編成**: 軽空母 3隻 ＋ 駆逐 3隻（または軽空母3 ＋ 駆逐2 ＋ 潜水艦1）
*   **ルート**: C → B（通常/輸送艦） → A（ボーキ獲得） → 撤退
*   **装備**: 駆逐艦にはドラム缶や大発動艇をフル装備
*   **特徴**: 空母系3隻を編成することでスタートからBマス（輸送ワ級が確実に出現するマス）へ固定されます。Bマスでデイリー/ウィークリーの輸送艦撃破任務を消化後、Aマスでボーキサイトを回収して撤退します（バシクル）。潜水艦をデコイとして1隻入れておくと駆逐艦の大破を防ぎやすくなります。
