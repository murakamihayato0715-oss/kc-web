## 【Extra Operation】5-5. サーモン海域北方

### 1. 海域概要・特徴
通常海域の中でも最凶の難易度を誇るExtra Operation（EO）海域です。
*   **作戦名**: 第二次サーモン海戦
*   **難度**: ☆☆☆(☆×12)
*   **クリア条件**: ボス旗艦を5回撃破（戦力ゲージは毎月1日の0:00に自動リセット）
*   **海域クリア報酬**: 勲章×1、特別戦果＋200
*   **特徴**:
    *   道中のPマスおよびボスマスに強敵「戦艦レ級elite」が出現します。レ級eliteは開幕航空戦に加えて、甲標的による超強力な「先制雷撃」を放ち、大和型すら一撃大破させる火力を持つため、非常に事故率が高い海域です。
    *   ゲージ破壊前後でボス編成が劇的に変化し、破壊後は「南方棲戦姫」が出現しなくなるなど大幅に難易度が下がります。このため、編成指定任務などはゲージ破壊後に挑戦するのがセオリーです。

### 2. ルート分岐法則
能動分岐（Fマス）や、戦艦・空母の数、索敵スコア（分岐点係数2）によってルートが決定されます。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **スタート** | ・駆逐4隻以上、ドラム缶搭載艦4隻以上、または大発搭載艦4隻以上 → **A**（戦闘回避）<br>・それ以外は **B**（対潜） |
| **B（対潜）** | ・正規空母3隻以上、または(戦艦級+雷巡)4隻以上、または雷巡3隻以上、または駆逐1隻以下 → **K**（通常・強）<br>・それ以外は **F**（能動分岐） |
| **F（能動分岐）** | **D**（うずしお） または **J**（通常） を選択可能 |
| **K** | ・【最速統一】、または駆逐2隻以上を含む【高速+統一】 → **H**（通常）<br>・それ以外は **G**（夜戦） |
| **G（夜戦）** | **I**（戦闘回避） 確定 |
| **I** | ・(戦艦級+空母系)3隻（過不足なく）かつ 駆逐2隻以上 → **L**（うずしお）<br>・それ以外は **M**（夜戦・タ級×2） |
| **M（夜戦）** | ・(戦艦級+空母系)4隻以上、または駆逐1隻以下 → **L**（うずしお）<br>・それ以外は **O**（能動分岐前・索敵チェックへ） |
| **H** | ・【最速統一】 → **N**（通常）<br>・(戦艦級+空母系)4隻以上 → **P**（通常・レ級）<br>・駆逐1隻以下 → **L**（うずしお）<br>・それ以外は **N**（通常） |
| **P** | ・【最速統一】 → **S（ボス）**<br>・【高速+統一】かつ (戦艦級+空母系)5隻以下かつ潜水艦なし → **S（ボス）**<br>・【索敵スコアチェック（分岐点係数2）】<br>・索敵スコア **80以上** で **S（ボス）** 確定、73未満でQ（逸れ） |
| **O** | ・【索敵スコアチェック（分岐点係数2）】<br>・索敵スコア **66以上** で **S（ボス）** 確定、63未満でR（逸れ） |

### 3. おすすめ編成例
#### ◆ 上ルート重量編成（ゲージ削り・破壊向け）
*   **編成**: 戦艦2 ＋ 正規空母2 ＋ 重巡級（または軽巡/雷巡）2
*   **ルート**: B（対潜） → K（通常/ヲ級改） → P（通常/レ級elite） → S（ボス）（道中3戦）
*   **制空値**: 430以上（ボスのヲ級改パターンに対して航空優勢を確保）
*   **索敵スコア**: 分岐点係数2で **80以上**
*   **特徴**: 大型艦の圧倒的な火力で道中とボスをねじ伏せる編成です。Pマスのレ級eliteの先制雷撃を耐えるため、戦艦にはバルジを装備させて防御を固めるのが有効です。Q逸れを防ぐため、(戦+空)は4隻以下に抑えるのが基本です。

#### ◆ 中央下ルート・長陸奥（または大和武蔵）タッチ編成（任務・S勝利狙い）
*   **編成**: 戦艦2（長門改二＋陸奥改二、または大和改二＋武蔵改二） ＋ 正規空母2 ＋ 駆逐2
*   **ルート**: B（対潜） → F（能動） → D（うずしお） → H（通常） → N（通常） → O → S（ボス）（道中3戦＋うずしお）
*   **装備**: 駆逐艦の1隻には対潜先制爆雷（Bマス対策）、もう1隻に対空カットインを推奨。うずしお対策に電探を各艦に分散。
*   **索敵スコア**: 分岐点係数2で **66以上**
*   **特徴**: 各種単発任務で「駆逐2」を要求された場合の基本となる編成。ボスの強烈なレ級eliteや南方棲戦姫に対して、ボス戦（Sマス）で特殊攻撃（長陸奥タッチ等）を発動させて昼戦での壊滅を狙います。

### 6. 特記事項・ギミック
*   **戦艦レ級の脅威**: 道中Pマス・ボスマスに登場するレ級eliteは、180機という超大スロットの航空攻撃に加え、甲標的による強力な先制雷撃を放ちます。
*   **任務消化はゲージ破壊後推奨**: ボスの「南方棲戦姫」は耐久380・装甲188と非常に硬く、破壊後は出現しなくなるため、S勝利任務などはゲージ破壊後にやるのが非常に有利です。
