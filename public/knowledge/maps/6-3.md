## 6-3. グアノ環礁沖海域

### 1. 海域概要・特徴
出撃制限があり、水雷戦隊の練度が試される海域です。ボス（Jマス）旗艦の4回撃沈でクリアとなります。
*   **作戦名**: K作戦
*   **難度**: ☆☆☆☆☆☆☆☆☆
*   **クリア条件**: ボス旗艦4回撃破（戦力ゲージクリア後は復活なし）
*   **特徴**:
    *   出撃可能な艦種が **水上機母艦、軽巡洋艦、練習巡洋艦、駆逐艦** のみに制限されています。
    *   道中のG・Hマスで「航空偵察」が行われ、ボスマスで **B勝利以上** を収めることで「開発資材（釘）」やボーキサイトなどの資源が手に入ります。
    *   航空偵察での大成功（開発資材2〜4個獲得）を狙うために、索敵・爆装値の高い水爆や水偵の装備が重要となります。

---

### 2. ルート分岐法則
能動分岐の選択および、艦種の隻数制限によってルートが決定されます。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **スタート** | **A**（能動分岐） 確定 |
| **A（能動分岐）** | **B**（対潜） または **C**（対潜） を選択（通常は戦闘回避のあるCを選択） |
| **C（対潜）** | **E**（通常） 確定 |
| **E** | ・軽巡3隻以上、または水母2隻以上 → **F**（通常・逸れ）<br>・【最短固定条件】: **軽巡1隻以下 かつ 駆逐3隻以上** で **G**（航空偵察）<br>・それ以外 → 60%でF / 40%でG のランダム |
| **G（航空偵察）** | **H**（航空偵察） 確定 |
| **H** | ・【索敵スコアチェック（分岐点係数3）】<br>　・索敵スコア **38以上** で **J（ボス）** 確定進行、36未満でI（行き止まり・逸れ） |

**主な攻略ルート**
* **A → C → E → G → H → J (最短ルート・推奨)**: 水母1 ＋ 軽巡1 ＋ 駆逐4（または水母1 ＋ 軽巡1 ＋ 練巡1 ＋ 駆逐3）。道中2戦（対潜1＋水上1）でボスへ向かいつつ、航空偵察を2回行える最も安定した攻略・周回ルートです。
* **A → C → E → F → H → J (4戦ルート)**: 軽巡3 ＋ 駆逐2 ＋ 水母1 など。Gマスの航空偵察を行えず、戦闘が1戦（Fマス）増えるため、通常攻略や周回でのメリットはありません。

### 3. おすすめ編成例
#### ◆ 最短ルート周回編成（推奨・開発資材集め）
*   **編成**: 水上機母艦1 ＋ 軽巡洋艦1 ＋ 駆逐艦4
*   **ルート**: A → C（対潜） → E（通常） → G（航空偵察） → H（航空偵察） → J（ボス）（道中2戦）
*   **装備**: 
    *   **水母（日進甲/三隈改二特）**: 主砲×2 ＋ 甲標的 ＋ 水爆（瑞雲改二系や晴嵐など高索敵水爆）
    *   **軽巡（矢矧改二乙）**: 主砲×2 ＋ 甲標的 ＋ 水偵（零式水上観測機など） ＋ [増設]バルジ
    *   **駆逐艦**: 1〜2隻に対潜先制爆雷（Cマスのソ級対策）を装備させ、残りは主砲連撃や電探、バルジを装備。
*   **特徴**: 水母・軽巡の開幕雷撃と先制対潜で道中の大破率を下げつつ、高い索敵値を維持してボスマスでの「大成功」を狙う定番の周回編成です。

### 6. 特記事項・ギミック
*   **航空偵察大成功の条件**: 索敵・爆装の高い瑞雲や晴嵐などの水上爆撃機、紫雲などの水上偵察機、二式大艇などの飛行艇を複数装備し、航空偵察判定でスコアを「35.2以上」確保することで、開発資材大成功（釘2〜4個）が確定します。
*   **資材持ち帰りの条件**: ボス（Jマス）で **B勝利以上** を収める必要があります。C敗退以下だった場合は道中の航空偵察で得られた資材はすべて没収されます。
