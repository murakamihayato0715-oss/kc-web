## 【Extra Operation】5-6. ラバウル方面海域

### 1. 海域概要・特徴
通常海域における最高難易度（★×13）を誇る、Extra Operation（EO）の最終海域です。出撃には5-5のクリアが必要です。
*   **作戦名**: ラバウル空襲艦隊を討て
*   **難度**: ☆☆☆☆(☆×13)
*   **クリア条件**: トリプルゲージ（第1:輸送、第2:戦力、第3:戦力）をすべて破壊（ゲージは毎月1日の0:00に自動リセット）
*   **海域クリア報酬**: 工廠資源×1、開発資材×4、特別戦果＋225
*   **特徴**:
    *   通常海域初のトリプルゲージ構成で、実質的に「ミニイベント海域」に近い仕様です。
    *   第1ゲージではPT小鬼群や反跳爆撃を伴う空襲、第2・第3ゲージでは「戦艦レ級(elite)」や「南太平洋空母棲姫」といったイベントボスクラスの強敵と相みえるため、支援艦隊の活用が極めて有効です。

---

### 2. ルート分岐法則
ゲージの段階および編成艦種、速力によってスタート地点（1または2）とルートが変化します。

*   **スタート地点選択**:
    *   スタート2解放前、または「戦艦級2隻以上かつ低速艦1隻以上」もしくは「(水母＋揚陸)1隻以上」で **スタート1** から出撃。
    *   それ以外は **スタート2** から出撃。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **第1ゲージ（輸送）** | スタート1 → **A**（通常） → **C2**（PT） → **D**（空襲） → **E**（揚陸） → **G（第一ボス・輸送）**<br>※高速統一かつ「軽巡1 ＋ 駆逐4 ＋ 水母or航巡1」で道中潜水回避の最短ルート。 |
| **第2ゲージ（戦力）** | スタート2 → **I**（能動分岐） → **J**（通常） → **K**（対潜） → **L**（通常/ヲ級改） → **N（第二ボス・戦力）**<br>※高速統一かつ「正規空母2 ＋ 重巡級1 ＋ 軽巡1 ＋ 駆逐2」で最短ルート。 |
| **第3ゲージ（戦力）** | スタート2 → **I**（能動分岐） → **O**（能動分岐） → **Q**（通常） → **U**（通常/タ級） → **X**（空襲） → **Z（第三ボス・戦力）**<br>※高速統一かつ「戦艦2 ＋ 正規空母1 ＋ 軽巡1 ＋ 駆逐2」で最短ルート。 |

### 3. おすすめ編成例
#### ◆ 第1ゲージ（輸送）最短ルート編成
*   **編成**: 航巡（または軽巡）1 ＋ 水上機母艦1 ＋ 駆逐4
*   **ルート**: スタート1 → A（通常） → C2（PT） → D（空襲） → E（揚陸） → G（第一ボス・輸送）（道中3戦）
*   **速力**: **高速統一**（低速艦を起用する場合はスタート1から出撃可能ですが、準最短B経由になります）
*   **装備**: 駆逐艦や水母に「大発動艇」などの輸送装備を搭載。PT対策として「武装大発」「装甲艇(AB艇)」や見張員・機銃、空襲対策に対空カットインを装備。
*   **特徴**: ボスA勝利以上でTPを削る編成。道中のPT小鬼群（C2）および反跳爆撃空襲（D）が非常に危険なため、それぞれの対策装備が必須です。

#### ◆ 第2ゲージ（戦力）最短ルート編成
*   **編成**: 正規空母2 ＋ 重巡級1 ＋ 軽巡1 ＋ 駆逐2
*   **ルート**: スタート2 → I（能動） → J（通常） → K（対潜） → L（通常/ヲ級改） → N（第二ボス）（道中3戦）
*   **速力**: **高速統一**
*   **制空値**: 350以上（ボスのヲ級改パターンに対して航空優勢を確保）
*   **特徴**: 空母2隻の開幕航空攻撃で道中の安全を図りつつ、夜戦でボスの戦艦レ級を撃沈する編成です。対空カットイン艦を1隻編成に組み込むとLマスのダブルヲー戦が安定します。

#### ◆ 第3ゲージ（戦力）高速最短ルート編成
*   **編成**: 戦艦2 ＋ 正規空母1 ＋ 軽巡1 ＋ 駆逐2
*   **ルート**: スタート2 → I（能動） → O（能動） → Q（通常） → U（通常/タ級） → X（空襲） → Z（第三ボス）（道中4戦）
*   **速力**: **高速統一**（低速戦艦は要高速化）
*   **制空値**: 300以上（ボス拮抗を狙う）
*   **索敵スコア**: 分岐点係数2で **80以上**
*   **特徴**: ボス「南太平洋空母棲姫」と戦う最終ゲージの編成。ボス・随伴ともに非常に高耐久かつ高火力なため、道中支援および決戦支援（砲撃支援）の投入を強く推奨します。

### 6. 特記事項・ギミック
*   **ギミック解除**: 第1ゲージ（輸送）破壊後、対潜戦を経由してRマス（戦闘回避マス）に1回到達すると、スタート地点2および第2ゲージ用ルートが開放されます。
*   **PT小鬼群と反跳爆撃空襲**: 第1ゲージのC2マスのPT対策（武装大発や見張員・機銃）、およびDマスの反跳爆撃対策として、対空カットインを持つ秋月型などの防空艦の起用が推奨されます。
*   **第三ボスの強さ**: ボスの南太平洋空母棲姫は非常に硬く、随伴にレ級eliteを伴うなど通常海域中トップクラスの難度です。クリア後はレ級が消えるなど、著しく弱体化します。
