## 5-3. サブ島沖海域

### 1. 海域概要・特徴
道中のほとんどが「夜戦のみ」の戦闘マス（夜戦マス）で構成されている、通常海域屈指の難所です。クリア条件はボス（Qマス）旗艦の5回撃沈です。
*   **作戦名**: 第一次サーモン沖海戦
*   **難度**: ☆☆☆☆☆☆☆☆☆
*   **クリア条件**: ボス旗艦5回撃沈（戦力ゲージクリア後は復活なし）
*   **主な入手アイテム**: 鋼材（Hマス）、弾薬（Fマス）
*   **特徴**:
    *   夜戦マスでは防御装甲が意味をなさず、敵味方ともに強力な一撃を放ちます。道中を突破するために、夜戦装備（夜偵・照明弾など）と強力な道中支援（前衛支援）が推奨されます。
    *   ボス前（K→H）の分岐条件が厳しいため、特定の編成パターン（ドラム缶搭載艦や艦種制限など）を遵守する必要があります。

---

### 2. ルート分岐法則
能動分岐（Oマス）を経て、KからHへ羅針盤を固定する条件が重要です。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **スタート** | ・【高速+以上統一】 → **D**（戦闘回避）<br>・(戦艦級+空母系)3隻以上、または(戦艦級+空母系)2隻かつ低速含む → **C**（通常）<br>・それ以外は **D**（戦闘回避） |
| **D** | **G**（戦闘回避） 確定 |
| **G** | ・航空戦艦、正規空母、または潜水艦を含む → **J**（夜戦）<br>・駆逐0隻、軽空母2隻以上、または駆逐1隻 → **I**（夜戦）<br>・(低速)戦艦2隻以上 → **J**（夜戦）<br>・それ以外は **I**（夜戦） |
| **I（夜戦）** | ・軽空母を含む、または戦艦級2隻以上 → **J**（夜戦）<br>・駆逐3隻以上、または軽巡1隻 かつ 駆逐2隻以上 → **O**（能動分岐）<br>・重巡級4隻以上 かつ 軽巡1隻で **O**<br>・それ以外はランダム |
| **J（夜戦）** | ・(戦艦級+空母系)4隻以上、または正規空母含む → **M**（空母戦・逸れ）<br>・それ以外はL（うずしお）またはN（夜戦）のランダム |
| **O（能動分岐）** | **K**（夜戦・タ級） または **P**（夜戦・輸送） を選択可能（通常はボス方面のKを選択） |
| **K（夜戦）** | ・駆逐4隻以上、または駆逐3隻 かつ 軽巡1隻で **H**（鋼材）<br>・駆逐2隻 かつ 【特定の条件】（高速+統一、航戦含む、ドラム缶2隻以上等）を満たせば **H**<br>・それ以外は **E**（逸れ） |
| **H（アイテム）** | **Q（ボス）** 確定 |

### 3. おすすめ編成例
#### ◆ 水上打撃編成（最短・推奨）
*   **編成**: 重巡（または航巡）5 ＋ 軽巡1
*   **ルート**: D → G → I（夜戦） → O（能動分岐） → K（夜戦） → H（鋼材） → Q（ボス）（道中2戦）
*   **装備**: 航巡に水戦を複数装備させて制空を確保。重巡級に「夜偵」「照明弾」を装備（探探灯は道中大破しやすいため非推奨）。
*   **特徴**: 夜戦に強い重巡級を主体とした最も安定する編成です。航巡の水戦でボスの制空拮抗〜優勢を取りつつ、夜戦連撃でボスを撃破します。

#### ◆ 戦艦入り最短編成（特殊攻撃活用）
*   **編成**: 高速戦艦2 ＋ 軽巡1 ＋ 駆逐2 ＋ 航巡1
*   **ルート**: D → G → I（夜戦） → O（能動分岐） → K（夜戦） → H（鋼材） → Q（ボス）（道中2戦）
*   **速力**: **高速統一**（低速戦艦を起用する場合は高速化が必須）
*   **装備**: K→H固定条件のため、駆逐2隻にそれぞれ「ドラム缶」を1個以上（艦隊合計2個以上）装備させること。
*   **特徴**: 戦艦の僚艦夜戦突撃（金剛型改二丙など）や特殊攻撃を活用して道中の夜戦を強引に突破し、ボスマスで戦艦の強力な昼火力で随伴を殲滅する編成です。

### 6. 特記事項・ギミック
*   **強力な道中支援の推奨**: 夜戦マスでの事故を防ぐため、前衛支援（砲撃支援）を出すことで大破撤退率を大きく下げられます。
*   **夜戦装備の選択**: 「九八式水上偵察機(夜偵)」や「照明弾」が効果的です。「探照灯」は装備した艦の回避率が極端に低下するため、道中夜戦が連続する本海域では装備させない方が無難です。
*   **ボスの南方棲戦姫**: 耐久380・装甲188と非常に硬く、夜戦での撃破が前提です。高夜戦火力の重巡や戦艦を艦隊の後方に配置（フィニッシャー化）しましょう。
*   **5-3-Pレベリング**: 能動分岐でPマス（輸送船団）へ進み、夜戦2戦を行って撤退するレベリング。Pマスの基礎経験値が700〜750と非常に高いため、高効率での育成が可能です。
