## 6-2. MS諸島沖

### 1. 海域概要・特徴
道中に強力な戦艦や空母が多数出現する、中〜上級者向けの海域です。クリア条件はボス（Kマス）旗艦の3回撃沈です。
*   **作戦名**: MS諸島防衛戦
*   **難度**: ☆☆☆☆☆☆☆☆☆
*   **クリア条件**: ボス旗艦3回撃破（戦力ゲージクリア後は復活なし）
*   **特徴**:
    *   道中のほぼ全戦闘マスに空母または戦艦が出現し、かつ対空砲火の強力な「軽巡ツ級」が確定出現するため、艦載機の撃墜対策や対空カットイン艦の編成が非常に重要になります。
    *   通常は、うずしおを回避する「南3戦最短ルート（CEJK）」で攻略するのが一般的です。通常海域で唯一、軽巡「酒匂」がボスドロップします。

---

### 2. ルート分岐法則
艦隊の重量（戦艦・空母の数）や、軽巡・駆逐の隻数、速力および索敵スコア（分岐点係数3）によってルートが決定されます。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **スタート** | ・(軽巡+駆逐)4隻以上 → **B**（通常/対潜）<br>・(航空戦艦+航巡+水母+揚陸艦)1隻以下 かつ 潜水艦4隻以下の場合：<br>　・(戦艦級+空母系)5隻以上 → **B**<br>　・(戦艦級+空母系)4隻 → B寄り(65%)のランダム<br>・それ以外は **C**（通常） |
| **B** | ・(軽巡+駆逐)5隻以上 → **D**（うずしお）<br>・空母系2隻以下 かつ 戦艦級0隻 → C寄り(70%)のランダム<br>・それ以外は **C**（通常） |
| **C** | ・潜水艦6隻、または(戦艦級+空母系)5隻以上、または(戦艦+空母+重巡)6隻 → **A**（戦闘回避・逸れ）<br>・(戦艦級+空母系)2隻以下 → **E**（通常）<br>・それ以外は **D**（うずしお） |
| **D（うずしお）** | ・戦艦級または空母系を含む、または駆逐2隻以下、または(軽巡+駆逐)4隻以下 → **F**（通常/空母）<br>・それ以外は **H**（通常/戦艦） |
| **E** | ・戦艦級2隻以上、空母系2隻以上、または駆逐1隻以下 → **F**（通常/空母）<br>・【索敵スコアチェック（分岐点係数3）】<br>　・索敵スコア **50以上** で **J**（通常/戦艦） 確定進行、43未満でI（通常/戦艦） |
| **F** | **I**（通常/戦艦） 確定 |
| **H** | ・【索敵スコアチェック（分岐点係数3）】<br>　・索敵スコア **32以上** で **K（ボス）** 確定、32未満でG（逸れ） |
| **I** | ・潜水艦4隻以上 → G（逸れ）<br>・【索敵スコアチェック（分岐点係数3）】<br>　・索敵スコア **40以上** で **K（ボス）** 確定、35未満でG（逸れ） |
| **J** | **K（ボス）** 確定 |

### 3. おすすめ編成例
#### ◆ 南3戦最短ルート編成（推奨・初攻略）
*   **編成**: 戦艦1 ＋ 正規空母1 ＋ 駆逐2 ＋ 雷巡1 ＋ 航巡1
*   **ルート**: C（通常） → E → J（通常/戦艦） → K（ボス）（道中3戦・うずしお回避）
*   **速力**: 高速統一不要（低速戦艦をそのまま起用可能）
*   **制空値**: 160以上（Jマスのヲ級改やボスのヲ級に対して航空優勢を確保）
*   **索敵スコア**: 分岐点係数3で **50以上**
*   **特徴**: 最も主流となる攻略編成です。雷巡の開幕雷撃で敵を減らし、航巡の水戦等で制空を稼ぎます。駆逐艦の1隻には対空カットインを装備させて空襲被害を抑えましょう。

#### ◆ 南4戦重量編成（駆逐艦なし・道中安定重視）
*   **編成**: 正規空母2 ＋ 重巡（または航巡）2 ＋ 雷巡1 ＋ 軽巡1
*   **ルート**: C → E → F（通常/空母） → I（通常/戦艦） → K（ボス）（道中4戦・うずしお回避）
*   **制空値**: 260以上（Fマスのヲ級flagship×2に対して航空優勢を狙う）
*   **索敵スコア**: 分岐点係数3で **40以上**
*   **特徴**: 駆逐艦を一切編成せず、耐久・装甲が高い艦だけで構成された重量編成です。道中のFマスが追加されるためボーキサイトの消費は多くなりますが、道中撤退率は抑えられます。

### 6. 特記事項・ギミック
*   **ツ級による艦載機撃墜対策**: ほぼ全戦闘マスに軽巡ツ級が出現するため、対空射撃回避能力を持つ艦載機（天山(村田隊)や爆戦岩井隊など）を大きめのスロットに搭載して、空母が行動不能になるのを防いでください。
*   **1戦撤退レベリング**: 戦艦や空母などを5隻以上編成して出撃すると、スタートからB→Cと進むルート（Bは「気のせいだった」、Cはル級等と戦闘）となり、CマスでのS勝利経験値が非常に高い（旗艦756・随伴504）ため、1戦撤退レベリングとして利用できます。
