## 5-4. サーモン海域

### 1. 海域概要・特徴
南方海域の通常海域ラストマップ。クリア条件はボス（Pマス）旗艦の5回撃沈です。
*   **作戦名**: 東京急行
*   **難度**: ☆(☆×10)
*   **クリア条件**: ボス旗艦5回撃沈（戦力ゲージクリア後は復活なし）
*   **主な入手アイテム**: 弾薬（Iマス）
*   **特徴**:
    *   空母を交えた強力な機動部隊で攻略する「上ルート」と、ドラム缶等の輸送装備を搭載して水雷戦隊で夜戦マスを駆け抜ける「中央最短ルート」が存在します。
    *   ボスマスは敵旗艦の輸送ワ級flagshipが比較的柔らかいですが、随伴に空母ヲ級flagship（新型機）や戦艦ル級/タ級flagshipが出現するため、ある程度の制空値と火力が求められます。

---

### 2. ルート分岐法則
空母の有無、ドラム缶の搭載数、駆逐・軽巡の数、速力および索敵スコアによってルートが決定されます。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **スタート** | ・空母系を含む → **B**（戦闘回避）<br>・戦艦級3隻以上、または重巡級5隻以上 → **A**（戦闘回避）<br>・ドラム缶/大発搭載艦の合計数が5隻以上（1隻の複数装備可） → **B**<br>・駆逐4隻以上、または軽巡1隻 かつ 駆逐3隻以上 → **B**<br>・それ以外は **A**（戦闘回避） |
| **A** | ・潜水艦を含む、または戦艦級5隻以上、または駆逐2隻以上、または重巡級3隻以上 → **D**（うずしお）<br>・それ以外は **F**（夜戦） |
| **B** | ・空母系または潜水艦を含む → **C**（通常/対潜）<br>・戦艦級を含み、かつ【低速】を含む → **D**（うずしお）<br>・(航空戦艦+(低速)戦艦)2隻以上 → **D**<br>・【高速+以上統一】、駆逐4隻以上、または軽巡1隻 かつ 駆逐3隻以上 → **E**（通常）<br>・それ以外はDまたはEのランダム |
| **C** | **G**（通常） 確定 |
| **D（うずしお）** | ・潜水艦を含む、または低速戦艦2隻以上、または戦艦級3隻以上 → **F**（夜戦）<br>・駆逐2隻以上 → **E**（通常）<br>・それ以外は **F**（夜戦） |
| **E** | **H**（夜戦） 確定 |
| **H（夜戦）** | **I**（弾薬マス） 確定 |
| **I（アイテム）** | **J**（夜戦） 確定 |
| **J（夜戦）** | **M**（戦闘回避） 確定 |
| **G** | ・正規空母2隻以下 → **L**（通常）<br>・それ以外は K（うずしお）寄りのランダム |
| **L** | ・【索敵スコアチェック（分岐点係数2）】<br>・索敵スコア **60以上** で **P（ボス）** 確定進行、56未満でN逸れ |
| **M** | ・【索敵スコアチェック（分岐点係数2）】<br>・索敵スコア **45以上** で **P（ボス）** 確定進行、41未満でO逸れ |

### 3. おすすめ編成例
#### ◆ 上ルート機動部隊編成（初攻略・安定重視）
*   **編成**: 正規空母2 ＋ 戦艦1 ＋ 重巡級1 ＋ 軽巡1 ＋ 駆逐1
*   **ルート**: B → C（通常/対潜） → G（通常/空母） → L（通常/タ級） → P（ボス）（道中3戦・上ルート最短）
*   **制空値**: 320以上（Gマスのヲ級flagship×2に対して航空優勢を狙う）
*   **索敵スコア**: 分岐点係数2で **60以上**
*   **特徴**: 空母の開幕爆撃と昼戦の弾着観測射撃により、安定してボスS勝利を狙える定番編成です。道中の軽巡ツ級対策として、対空射撃回避能力を持つ艦載機を装備させるのが有効です。

#### ◆ 中央最短ルート編成（高速統一・周回向け）
*   **編成**: 高速戦艦（または重巡級/軽巡）2 ＋ 軽巡1 ＋ 駆逐3（要高速統一。ドラム缶/大発の合計5個以上）
*   **ルート**: B → E（通常） → H（夜戦・リ級） → I（弾薬） → J（夜戦・輸送） → M → P（ボス）（道中3戦）
*   **速力**: **高速統一**（夕張改二特などの低速艦を起用する場合は高速化が必須）
*   **装備**: 夜戦装備（夜偵、照明弾）を推奨。
*   **索敵スコア**: 分岐点係数2で **45以上**
*   **特徴**: ドラム缶/大発系を合計5個以上搭載してルートを固定します。夜戦マスを2つ通るため事故の危険はありますが、ボーキサイトを消費せず、ボスで輸送ワ級などの補給艦を多数狩れるためウィークリー任務や戦果稼ぎに向いています。伊勢型改二（高速化）で制空拮抗・優勢を取るのが主流です。

### 6. 特記事項・ギミック
*   **索敵スコアチェック**: ボス前の分岐（L→P、M→P）では索敵スコアが要求されます。上ルート（L→P）は係数2で60以上、中央ルート（M→P）は係数2で45以上が必要です。
*   **制空補助**: 空母を編成できない中央ルートで弾着観測射撃を使用したい場合、水戦を装備した航巡や、高速化した伊勢型改二に艦戦を搭載して制空拮抗・優勢を確保してください。
