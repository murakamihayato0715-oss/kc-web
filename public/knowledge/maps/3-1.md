## 3-1. モーレイ海

### 1. 海域概要・特徴
北方海域の最初の海域。3-1からはflagship級の敵が多く出現するようになります。
*   **作戦名**: モーレイ海哨戒
*   **難度**: ☆☆☆☆
*   **クリア条件**: ボスB勝利以上
*   **主な入手アイテム**: 弾薬
*   **特徴**: 敵は水上艦中心で航空戦力は貧弱です。制空値約90で全マス制空権確保が可能。一方、**重巡リ級flagship**が多く、約80の火力は駆逐に大破を与えやすく注意が必要です。道中2戦の最短ルートが推奨されますが、超重量編成（戦艦・空母のみ）はボス前でEマスに逸れるため編成に注意が必要です。

---

### 2. ルート分岐法則
艦隊の戦艦・空母の数と駆逐の数でルートが決まります。

| 分岐点 | 条件（上から優先） |
| --- | --- |
| **C** | (駆逐+海防)1以下 → D<br>(航戦+水母+補給+潜母+軽巡)3以上 → 50%でB<br>(戦艦+空母)3以上 → D<br>(水母+補給+潜母)0 → F<br>(駆逐+海防)2以上 → F<br>その他 → 50%でB / 50%でF |
| **D** | (戦艦+空母)5以上 → E（逸れ）<br>潜水6 → E<br>潜母1かつ潜水5 → G<br>その他 → F |
| **F** | G固定 |

**ルート固定条件**

| 編成 | ルート | 条件 |
| --- | --- | --- |
| (戦艦+空母)2・(駆逐+海防)2・自由2 | CFG（道中2戦・推奨） | (航戦+軽巡+水母+補給)2以下に注意 |
| (戦艦+空母)3〜4・自由2 | CDFG（道中3戦） | 戦艦+空母が5以上になるとEに逸れる |
| 潜母1・潜水5 | CDG | 潜水特化専用編成 |

### 3. おすすめ編成例
#### ◆ 初攻略・通常周回（CFGルート・推奨）
*   **編成**: 正規空母2 ＋ 重巡1 ＋ 雷巡1 ＋ 駆逐2
*   **ルート**: C → F → G（ボス）（道中2戦）
*   **制空値**: 約94（目安90以上、確保84 優勢42）
*   **陣形**: 全戦単縦陣
*   **特徴**: (航戦+軽巡+水母+補給)を3以上にするとB側に逸れるため注意。空母は戦艦に変更可。雷巡がいなければ重巡で代替可。

#### ◆ 重量押し切り編成（CDFGルート・道中3戦）
*   **編成**: (戦艦+空母)3〜4 ＋ 自由2
*   **ルート**: C → D → F → G（ボス）（道中3戦）
*   **特徴**: (戦艦+空母)が5以上になるとEマスに逸れるため4以下を維持すること。

### 6. 特記事項・ギミック
*   ギミックなし。クリア条件はボスB勝利以上と易しめ。
*   2-4を突破できた提督であれば問題なくクリア可能。重要な海域ではないため、クリアだけして次に進んでOK。
*   **重巡リ級flagship対策**: 火力約80で駆逐に大破を与えやすい。駆逐艦主体の任務では自由枠に空母か戦艦を入れることを推奨。
